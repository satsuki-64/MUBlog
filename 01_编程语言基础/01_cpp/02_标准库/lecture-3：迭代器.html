Lecture 3：迭代器

使用方法

基础概念

概念：C++ 中的迭代器

  • 概念：
    • 在 C++ 中，​迭代器（Iterator）​ 是一种抽象的设计模式，用于提供对容器（如 vector、list、map 等）中元素的统一访问方式；
    • 它类似于指针，允许遍历容器中的元素，而无需关心容器的底层实现细节；
    • 迭代器是 STL 的核心组件之一，与算法（Algorithms）和容器（Containers）紧密结合；
    • 迭代器有有效和无效之分，这一点和指针差不多，有效的迭代器或者指向某个元素，或者指向容器中尾元素的下一位置，其他所有情况都属于无效；
  • 使用：
      1. 获得迭代器：
      • 容器提供以下成员函数获取迭代器；
      • begin()：返回指向第一个元素的迭代器；
      • end()：返回指向“尾后元素”（最后一个元素的下一个位置）的迭代器；
      • cbegin(), cend()：常量迭代器，只读；
      • rbegin(), rend()：反向迭代器，从尾到头遍历；
      • 注意：如果容器为空，则 begin 和 end 返回的是同一个迭代器，都是尾后迭代器；
      1. 使用迭代器与算法：
      • STL 算法通过迭代器操作容器；
      • 比如：
        • std::vector<int> v1 = {2,1,3};
        • std::sort(v1.begin(),v1.end());
      • 这里代码就是使用迭代器与算法；

核心操作

概念：迭代器核心操作

  • 概念：*item
    • 概念：
      • 表示返回迭代器 item 所指向的元素的引用；
      • 解引用迭代器可获得迭代器所指的对象，如果该对象的类型恰好是类，就有可能希望进一步访问它的成员；
    • 举例：
      • 正确：(*it).empty() // 解引用 it，然后调用结果对象的 empty 成员
      • 错误：*it.empty() // 错误: 试图访问 it 的名为 empty 的成员，但 it 是个迭代器
  • 概念：item->mem
    • 表示解引用 item 并获取该引用所命名为 mem 的成员；
    • 其作用是将解引用和成员访问两个操作结合在一起；
    • 即等价于 (*item).mem；
  • 概念：移动
    • 使用：
      • ++item
    • 作用：
      • 迭代器指向下一个元素；
    • 注意：
      • 因为 end 返回的迭代器并不实际指示某个元素，所以不能对其进行递增或解引用的操作；
  • 概念：item1 == item2
    • 判断两个迭代器是否相等 (不相等)；
    • 如果两个迭代器指示的是同一个元素或者它们是同一个容器的尾后迭代器，则相等。反之，不相等；
  • 注意：
    • 谨记，但凡是使用了迭代器的循环体，都不要向迭代器所属的容器添加元素；

begin 与 end 运算符

概念：begin 与 end 运算符

  • 概念：
    • begin 和 end 返回的具体类型由对象是否是常量决定；
    • 如果对象是常量，begin 和 end 返回 const iterator；
    • 如果对象不是常量，返回 iterator；
  • 概念：cbegin 和 cend
    • 如果对象只需读操作而无须写操作的话，最好使用常量类型 (比如 const iterator)；
    • 为了便于专门得到 const iterator 类型的返回值，C++11 新标准引入了两个新函数，分别是 cbegin 和 cend；
    • 比如：auto it3=v.cbegin(); //it3的类型是 vector<int>::const iterator

迭代器运算

迭代器算术运算

概念：迭代器算术运算

  • 概念：
    • 可以令迭代器和一个整数值相加 (或相减)，其返回值是向前 (或向后) 移动了若干个位置的迭代器；
    • 比如：iter + n 表示迭代器加上一个整数值仍得一个迭代器，迭代器指示的新位置与原来相比向前移动了若干个元素。结果迭代器或者指示容器内的一个元素，或者指示容器尾元素的下一位置；
    • 比如：iter - n 表示迭代器减去一个整数值仍得一个迭代器，迭代器指示的新位置与原来相比向后移动了若干个元素。结果迭代器或者指示容器内的一个元素，或者指示容器尾元素的下一位置；
  • 举例：得到中间位置的元素
    • auto mid = *(v1.begin() + v1.size()/2);
  • 概念：difference_type
    • 只要两个迭代器指向的是同一个容器中的元素或者尾元素的下一位置，就能将其相减，所得结果是两个迭代器的距离；
    • 所谓距离指的是右侧的迭代器向前移动多少位置就能追上左侧的迭代器，其类型是名为 difference_type 的带符号整型数；
    • string 和 vector 都定义了 difference_type，因为这个距离可正可负，所以其是带符号类型的；

自定义迭代器

概念：自定义迭代器

  • 概念：
    • 在 C++ 中，​自定义迭代器允许你为自定义的容器或类型对象提供类似 STL 迭代器的访问接口，使其能够与标准库算法（如 std::sort、std::find）无缝协作；
  • 实现：
      1. 在容器类中定义迭代器类；
      • 需要实现以下功能：
      • 迭代器类型别名​（如 value_type、iterator_category）；
      • 解引用操作符​（operator*）访问元素；
      • 递增/递减操作符​（operator++、operator--）；
      • 比较操作符​（operator!= 或 operator==）；
      1. 在容器中提供 begin() 和 end()
      • 容器类需要提供成员函数返回迭代器；
      1. 使用迭代器；
  • 举例：
#include <iostream>
#include <algorithm> // 用于 std::sort

template <typename T>
class SimpleVector {
    T* data;
    size_t size;
public:
    // 自定义迭代器类
    class Iterator {
        T* ptr;
    public:
        // 必须定义的迭代器类型别名（STL 兼容性）
        using iterator_category = std::random_access_iterator_tag;
        using value_type = T;
        using difference_type = std::ptrdiff_t;
        using pointer = T*;
        using reference = T&;

        // 构造函数
        Iterator(T* p) : ptr(p) {}

        // 解引用
        reference operator*() const { return *ptr; }
        pointer operator->() { return ptr; }

        // 算术运算（随机访问迭代器）
        Iterator& operator++() { ++ptr; return *this; }    // 前置++
        Iterator operator++(int) { Iterator tmp = *this; ++ptr; return tmp; } // 后置++
        Iterator& operator--() { --ptr; return *this; }    // 前置--
        Iterator operator+(difference_type n) const { return Iterator(ptr + n); }
        Iterator operator-(difference_type n) const { return Iterator(ptr - n); }
        difference_type operator-(const Iterator& other) const { return ptr - other.ptr; }

        // 比较运算符
        bool operator==(const Iterator& other) const { return ptr == other.ptr; }
        bool operator!=(const Iterator& other) const { return ptr != other.ptr; }
        bool operator<(const Iterator& other) const { return ptr < other.ptr; }
    };

    // 容器构造函数/析构函数
    SimpleVector(size_t n) : size(n), data(new T[n]) {}
    ~SimpleVector() { delete[] data; }

    // 提供 begin() 和 end()
    Iterator begin() { return Iterator(data); }
    Iterator end() { return Iterator(data + size); }

    // 支持下标访问
    T& operator[](size_t index) { return data[index]; }
};

int main() {
    SimpleVector<int> vec(5);
    for (size_t i = 0; i < 5; ++i) {
        vec[i] = 5 - i; // 赋值为 5, 4, 32, 1
    }

    // 使用迭代器遍历
    for (auto it = vec.begin(); it != vec.end(); ++it) {
        std::cout << *it << " "; // 输出：5 4 3 2 1
    }

    // 使用 STL 算法排序
    std::sort(vec.begin(), vec.end());

    // 排序后输出
    for (auto num : vec) { // 范围 for 循环（依赖 begin()/end()）
        std::cout << num << " "; // 输出：1 2 3 4 5
    }

    return 0;
}
