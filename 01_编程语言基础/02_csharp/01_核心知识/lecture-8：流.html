Lecture 8：流

.NET 流架构

概念：.NET 流架构

  • 概念：
    • .NET 流的架构主要包含三个概念：后台存储、装饰器以及流适配器；
      1. 后台存储流：数据读写的端点或源头
      • 提供最基础的字节序列读写能力，与具体存储介质（如文件、网络、内存）直接交互；
      1. 装饰器流：功能的增强者
      • 包装另一个流，透明地提供数据转换功能（如压缩、加密），​不改变流的基本字节操作接口；
      • 因为有装饰器流，因此后台存储流无需自己实现压缩或者加密的方法，并且装饰器类之间可以串连不同的装饰器流，为某个简单的后台存储流提供复杂且可定制的功能；
      1. 流适配器：数据表示的转换者
      • 将底层的字节流转换为更高级别的数据类型（如文本、XML、基本数据类型），​隐藏了字节级别的细节；
      • 后台存储流和装饰器流仅支持字节处理。虽然这种方式灵活又高效，但应用程序往往需要处理更高层次的数据，例如文本或者 XML。适配器正好弥补了这个鸿沟。它在类中创建专门的方法以支持特定的格式；
      • 例如，TextReader 有一个 ReadLine 方法，而 XmlTextWriter 则拥有 WriteAttribute 方法；
  • 特点：
    • 这种架构的核心优势在于其组合性和单一职责原则；
    • 可以通过简单的对象组合（将一个对象传入另一个对象的构造函数）来构建功能强大的流处理链；
  • 图示：
  • 示例：
// 1. 创建后台存储流 -> 文件
using (FileStream fileStream = new FileStream("data.txt.gz", FileMode.Open))
{
    // 2. 添加装饰器功能 -> 解压缩
    using (GZipStream decompressor = new GZipStream(fileStream, CompressionMode.Decompress))
    {
        // 3. 添加适配器，以文本形式读取 -> 文本读取
        using (StreamReader reader = new StreamReader(decompressor))
        {
            string line;
            while ((line = reader.ReadLine()) != null)
            {
                Console.WriteLine(line);
            }
        }
    }
}

概念：Stream 流基础概念

  • 概念：
    • Stream 是所有流对象的基类，其通过了若干流的核心操作方法；
  • 总结：
  • 概念：核心方法
    • Read/Write：参数一是当前要读取或者写入的字节数组，参数二是当前读取的偏移量，参数三是读取的数量；
  • 补充：
    • 从 .NETFramework 4.5 开始，它还提供了异步的 Read 和 Write 方法，这些方法都会返回 Task 对象，并支持取消令牌；

后台存储流

后台存储流构成

概念：后台存储流构成

  • 概念：
    • 后台存储流是流架构中的端点或源头。它们是直接与特定物理（或逻辑）存储介质进行字节级交互的 Stream 类的具体实现。它们是整个数据流操作的基石，所有高级功能（如装饰器、适配器）都构建在它们之上；
  • 概念：什么是可查找
    • 可查找流​意味着你可以通过 Seek() 方法或设置 Position 属性，随意地在数据流中跳转（例如，从开头跳到末尾，或从中间某个位置继续读取）。​文件和内存天然支持这种随机访问，网络和管道数据是顺序到达的，因此不支持查找，因此 NetworkStream 和 PipeStream 不支持；

MemoryStream 类

概念：MemoryStream 类

  • 概念：
    • MemoryStream 使用数组作为后台存储，这在一定程度上与使用流的目的是相违背的，因为这个后台存储都必须一次性地驻留在内存中；
    • 然而，MemoryStream 仍然有一定的用途，例如随机访问一个不可查找的流；
    • 如果原始的流的大小可以承受，则可以通过如下的方式将其复制到 MemoryStream 中：
      • var ms= new MemoryStream ();
      • sourceStream.CopyTo (ms);
    • 我们可以调用 ToArray 方法将一个 MemoryStream 转换为一个字节数组，而 GetBuffer 方法则更加高效地直接返回底层存储数组的引用；
    • 但是，需要注意的是这个数组通常会比流的实际长度要长一些。
  • 补充：关于 MemoryStream 实例的刷新和关闭
    • MemoryStream 的关闭和刷新不是必须的，如果关闭了一个 Memory-Stream 就无法再次读写了。但是我们仍然可以调用 ToArray 方法来获得底层的数据。而刷新操作则不会对内存流执行任何操作；
  • 代码：
using UnityEngine;
using System.IO;
using System.Text;
using System.Runtime.Serialization.Formatters.Binary;

public class MemoryStreamDemo : MonoBehaviour
{
    [System.Serializable]
    public class PlayerData
    {
        public string playerName;
        public int level;
        public float health;
        public Vector3 position;
        
        public PlayerData(string name, int lvl, float hp, Vector3 pos)
        {
            playerName = name;
            level = lvl;
            health = hp;
            position = pos;
        }
    }
    
    void Start()
    {
        DemoBasicMemoryStream();
        DemoBinarySerialization();
        DemoCustomDataTypes();
    }
    
    /// <summary>
    /// 演示1: MemoryStream的基本读写操作
    /// 展示如何向内存流写入数据并从内存流读取数据
    /// </summary>
    void DemoBasicMemoryStream()
    {
        Debug.Log("=== 基本MemoryStream操作演示 ===");
        
        // 创建MemoryStream对象 - 在内存中创建数据流[4](@ref)
        using (MemoryStream stream = new MemoryStream())
        {
            // 示例数据：玩家信息字符串
            string playerInfo = "Player:John,Level:10,Health:85.5";
            
            // 将字符串转换为字节数组并写入内存流
            byte[] dataBytes = Encoding.UTF8.GetBytes(playerInfo);
            stream.Write(dataBytes, 0, dataBytes.Length);
            
            Debug.Log($"流容量: {stream.Capacity}, 流长度: {stream.Length}, 当前位置: {stream.Position}");
            
            // 重置流位置到开头以便读取
            stream.Seek(0, SeekOrigin.Begin);
            
            // 从内存流读取数据
            byte[] readBuffer = new byte[stream.Length];
            // 将当前 stream 中的数据读取到字节数组 readBuffer 当中
            stream.Read(readBuffer, 0, (int)stream.Length);
            string decodedString = Encoding.UTF8.GetString(readBuffer);
            
            Debug.Log($"读取的数据: {decodedString}");
        } // using语句自动释放资源
    }
    
    /// <summary>
    /// 演示2: 使用MemoryStream进行二进制序列化
    /// 将Unity自定义类对象序列化为二进制数据
    /// </summary>
    void DemoBinarySerialization()
    {
        Debug.Log("=== 二进制序列化演示 ===");
        
        // 创建玩家数据对象
        PlayerData player = new PlayerData("UnityPlayer", 25, 75.5f, new Vector3(10, 5, 20));
        
        // 序列化：对象 → 字节数组
        byte[] serializedData = SerializeToBytes(player);
        Debug.Log($"序列化后的字节数组长度: {serializedData.Length}");
        
        // 反序列化：字节数组 → 对象
        PlayerData deserializedPlayer = DeserializeFromBytes<PlayerData>(serializedData);
        Debug.Log($"反序列化结果 - 名称: {deserializedPlayer.playerName}, 等级: {deserializedPlayer.level}, 位置: {deserializedPlayer.position}");
    }
    
    /// <summary>
    /// 将对象序列化为字节数组
    /// 使用BinaryFormatter和MemoryStream组合[6](@ref)
    /// </summary>
    byte[] SerializeToBytes<T>(T obj) where T : class
    {
        // 使用using确保资源正确释放
        using (MemoryStream memoryStream = new MemoryStream())
        {
            BinaryFormatter formatter = new BinaryFormatter();
            
            try
            {
                // 将对象序列化到内存流[6](@ref)
                formatter.Serialize(memoryStream, obj);
                return memoryStream.ToArray(); // 获取字节数组
            }
            catch (System.Exception e)
            {
                Debug.LogError($"序列化失败: {e.Message}");
                return null;
            }
        }
    }
    
    /// <summary>
    /// 从字节数组反序列化对象
    /// </summary>
    T DeserializeFromBytes<T>(byte[] data) where T : class
    {
        if (data == null || data.Length == 0)
            return null;
            
        using (MemoryStream memoryStream = new MemoryStream(data))
        {
            BinaryFormatter formatter = new BinaryFormatter();
            
            try
            {
                // 从内存流反序列化对象[6](@ref)
                return (T)formatter.Deserialize(memoryStream);
            }
            catch (System.Exception e)
            {
                Debug.LogError($"反序列化失败: {e.Message}");
                return null;
            }
        }
    }
    
    /// <summary>
    /// 演示3: 自定义数据类型的读写
    /// 展示如何读写各种基本数据类型[1](@ref)
    /// </summary>
    void DemoCustomDataTypes()
    {
        Debug.Log("=== 自定义数据类型读写演示 ===");
        
        using (ExtendedMemoryStream stream = new ExtendedMemoryStream())
        {
            // 写入各种数据类型
            stream.WriteInt(42);
            stream.WriteFloat(3.14f);
            stream.WriteBool(true);
            stream.WriteString("Unity MemoryStream演示");
            stream.WriteVector3(new Vector3(1, 2, 3));
            
            // 重置位置准备读取
            stream.Seek(0, SeekOrigin.Begin);
            
            // 按写入顺序读取数据
            int intValue = stream.ReadInt();
            float floatValue = stream.ReadFloat();
            bool boolValue = stream.ReadBool();
            string stringValue = stream.ReadString();
            Vector3 vectorValue = stream.ReadVector3();
            
            Debug.Log($"读取结果: int={intValue}, float={floatValue}, bool={boolValue}");
            Debug.Log($"字符串: {stringValue}, 向量: {vectorValue}");
        }
    }
}

/// <summary>
/// 扩展的MemoryStream类，添加对Unity常用数据类型的支持[1](@ref)
/// </summary>
public class ExtendedMemoryStream : MemoryStream
{
    public ExtendedMemoryStream() : base() { }
    public ExtendedMemoryStream(byte[] buffer) : base(buffer) { }
    
    #region 基本数据类型读写
    public int ReadInt()
    {
        byte[] bytes = new byte[4];
        Read(bytes, 0, 4);
        return System.BitConverter.ToInt32(bytes, 0);
    }
    
    public void WriteInt(int value)
    {
        byte[] bytes = System.BitConverter.GetBytes(value);
        Write(bytes, 0, bytes.Length);
    }
    
    public float ReadFloat()
    {
        byte[] bytes = new byte[4];
        Read(bytes, 0, 4);
        return System.BitConverter.ToSingle(bytes, 0);
    }
    
    public void WriteFloat(float value)
    {
        byte[] bytes = System.BitConverter.GetBytes(value);
        Write(bytes, 0, bytes.Length);
    }
    
    public bool ReadBool()
    {
        return ReadByte() == 1;
    }
    
    public void WriteBool(bool value)
    {
        WriteByte((byte)(value ? 1 : 0));
    }
    
    public string ReadString()
    {
        int length = ReadInt();
        byte[] bytes = new byte[length];
        Read(bytes, 0, length);
        return Encoding.UTF8.GetString(bytes);
    }
    
    public void WriteString(string value)
    {
        byte[] bytes = Encoding.UTF8.GetBytes(value);
        WriteInt(bytes.Length);
        Write(bytes, 0, bytes.Length);
    }
    #endregion
    
    #region Unity特有类型支持
    public Vector3 ReadVector3()
    {
        float x = ReadFloat();
        float y = ReadFloat();
        float z = ReadFloat();
        return new Vector3(x, y, z);
    }
    
    public void WriteVector3(Vector3 vector)
    {
        WriteFloat(vector.x);
        WriteFloat(vector.y);
        WriteFloat(vector.z);
    }
    
    public Vector2 ReadVector2()
    {
        float x = ReadFloat();
        float y = ReadFloat();
        return new Vector2(x, y);
    }
    
    public void WriteVector2(Vector2 vector)
    {
        WriteFloat(vector.x);
        WriteFloat(vector.y);
    }
    #endregion
}
