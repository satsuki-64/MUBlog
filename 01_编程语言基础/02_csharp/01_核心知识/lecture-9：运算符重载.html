什么是运算符重载

运算符重载基础概念

概念：什么是运算符重载

  • 概念：
    • 运算符可以通过重载以更自然的语法操作自定义类型；
    • 下面的运算符符号都是可以重载的：
    • 除此之外，以下运算符也可以重载：
      • 隐式和显式转换（使用 implicit 和 explicit 关键字实现）；
      • true 而 false 运算符（非字面量）
    • 以下运算符可以间接重载：
      • 复合赋值运算符（例如：+= 和 -=）可以通过重写非复合运算符（例如 +、-）隐式重写；
      • 条件运算符 && 和 || 可以通过重写按位操作运算符 & 和 | 隐式重写；
  • 概念：运算符函数
    • 运算符是通过声明运算符函数进行重载的，运算符函数具有以下规则：
      1. 函数名为 operator，关键字跟上运算符符号；
      1. 运算符函数必须是 static 和 public 的；
      1. 运算符函数的参数即操作数；
      1. 运算符函数的返回类型表示表达式的结果；
      1. 运算符函数的操作数中，至少有一个类型和声明运算符函数的类型是一致；

运算符重载示例

概念：运算符重载示例

public struct Note
{
	public int valueInNote;
	public Note(int valueTemp)
	{
		valueInNote = valueTemp;
	}
	
	public static Note operator +(Note x, int semitones)
	{
		return new Note(x.valueinNote + semitones);
	}
}

// 使用 + 符号
Note A = new Note(2);
Note B = A + 2; // B.valueInNote = 4

// 使用 += 符号
// 重载运算符会自动支持相应的复合赋值运算符，因为我们重载了+号，所以自然就可以使用+=了
B += 2; // B.valueInNote = 6

重载等号和比较运算符

基础概念

概念：重载等号和比较运算符

  • 概念：
    • 通常在我们使用结构体（或类，但不常见）时需要重载等号和比较运算符；
  • 概念：要求与限制
    • 重载等号和比较运算符有一些特殊的规则和要求，在这里我们将其总结为：
    • （1）成对重载：C# 编译器要求逻辑上成对的运算符必须同时定义，这些运算符包括：
      • （==、!=)、（<、>）和（<=、>=）
      • 如果只定义一个，则会产生报错，比如：
    • （2）Equals 和 GetHashCode：
      • 在大多数情况下，如果重载了（==）和（!=）运算符，则通常也需要重载 object 中定义的 Equals 和 GetHashCode 方法，使之具有合理的行为。如果没有按照要求重载，则 C# 编译器会发出警告；
    • （3）IComparable 和 IComparable<T>：
      • 如果重载了（<、>）和（<=、>=）运算符，那么还应当实现 IComparable 和 IComparable<T> 接口；

代码示例

代码：成对重载示例（== 和 !=）

public class Vector2D
{
    public double X { get; set; }
    public double Y { get; set; }
    
    public Vector2D(double x, double y)
    {
        X = x;
        Y = y;
    }
    
    // 重载 == 运算符：比较两个Vector2D对象的内容是否相等
    public static bool operator ==(Vector2D v1, Vector2D v2)
    {
        // 如果两个引用指向同一个对象，或者都为null，则相等
        if (ReferenceEquals(v1, v2)) return true;
        
        // 如果其中一个为null，另一个不为null，则不相等
        if (v1 is null || v2 is null) return false;
        
        // 比较X和Y坐标的数值相等性（使用近似比较处理浮点数精度问题）
        return Math.Abs(v1.X - v2.X) < 0.0001 && Math.Abs(v1.Y - v2.Y) < 0.0001;
    }
    
    // 重载 != 运算符：必须与==运算符成对重载
    // 逻辑上直接取==运算符的反结果
    public static bool operator !=(Vector2D v1, Vector2D v2)
    {
        return !(v1 == v2);
    }
}

// 使用示例
Vector2D vec1 = new Vector2D(1.0, 2.0);
Vector2D vec2 = new Vector2D(1.0, 2.0);
Vector2D vec3 = new Vector2D(3.0, 4.0);

Console.WriteLine(vec1 == vec2);  // 输出：True
Console.WriteLine(vec1 == vec3);  // 输出：False
Console.WriteLine(vec1 != vec3); // 输出：True

代码：Equals 和 GetHashCode 重写示例

public class Student : IEquatable<Student>
{
    public string Name { get; set; }
    public int Age { get; set; }
    public string StudentId { get; set; }
    
    public Student(string name, int age, string studentId)
    {
        Name = name;
        Age = age;
        StudentId = studentId;
    }
    
    // 重载 == 运算符
    public static bool operator ==(Student s1, Student s2)
    {
        if (ReferenceEquals(s1, s2)) return true;
        if (s1 is null || s2 is null) return false;
        
        return s1.StudentId == s2.StudentId && s1.Name == s2.Name && s1.Age == s2.Age;
    }
    
    // 重载 != 运算符
    public static bool operator !=(Student s1, Student s2)
    {
        return !(s1 == s2);
    }
    
    // 重写Equals(object)方法：用于与任意对象比较
    public override bool Equals(object obj)
    {
        if (obj == null || !(obj is Student))
            return false;
            
        return Equals((Student)obj);
    }
    
    // 实现IEquatable<Student>接口：提供类型安全的比较
    public bool Equals(Student other)
    {
        if (other is null) return false;
        
        return this.StudentId == other.StudentId && 
               this.Name == other.Name && 
               this.Age == other.Age;
    }
    
    // 重写GetHashCode方法：必须与Equals方法保持一致
    // 如果两个对象Equals返回true，它们的GetHashCode必须返回相同的值
    public override int GetHashCode()
    {
        // 使用所有参与比较的字段计算哈希码
        unchecked // 允许整数溢出而不抛出异常
        {
            int hash = 17;
            hash = hash * 23 + (Name?.GetHashCode() ?? 0);
            hash = hash * 23 + Age.GetHashCode();
            hash = hash * 23 + (StudentId?.GetHashCode() ?? 0);
            return hash;
        }
    }
}

// 使用示例
Student student1 = new Student("张三", 20, "S1001");
Student student2 = new Student("张三", 20, "S1001");
Student student3 = new Student("李四", 22, "S1002");

Console.WriteLine(student1 == student2);     // 输出：True
Console.WriteLine(student1.Equals(student2)); // 输出：True
Console.WriteLine(student1.GetHashCode() == student2.GetHashCode()); // 输出：True

代码：IComparable 接口实现示例

public class Temperature : IComparable, IComparable<Temperature>
{
    public double Celsius { get; set; }
    
    public Temperature(double celsius)
    {
        Celsius = celsius;
    }
    
    // 重载 < 运算符
    public static bool operator <(Temperature t1, Temperature t2)
    {
        if (t1 is null) return t2 != null; // null小于任何非null值
        return t1.CompareTo(t2) < 0;
    }
    
    // 重载 > 运算符
    public static bool operator >(Temperature t1, Temperature t2)
    {
        if (t1 is null) return false; // null不大于任何值
        return t1.CompareTo(t2) > 0;
    }
    
    // 重载 <= 运算符
    public static bool operator <=(Temperature t1, Temperature t2)
    {
        if (t1 is null) return true; // null小于等于任何值（包括null）
        return t1.CompareTo(t2) <= 0;
    }
    
    // 重载 >= 运算符
    public static bool operator >=(Temperature t1, Temperature t2)
    {
        if (t1 is null) return t2 is null; // null只大于等于null
        return t1.CompareTo(t2) >= 0;
    }
    
    // 实现IComparable接口：提供标准的比较方法
    public int CompareTo(object obj)
    {
        if (obj == null) return 1; // 非null对象大于null
        
        if (obj is Temperature other)
            return CompareTo(other);
            
        throw new ArgumentException("比较的对象不是Temperature类型");
    }
    
    // 实现IComparable<Temperature>接口：提供类型安全的比较
    public int CompareTo(Temperature other)
    {
        if (other is null) return 1; // 非null大于null
        
        // 基于摄氏温度进行比较
        return Celsius.CompareTo(other.Celsius);
    }
    
    // 重写Equals和GetHashCode以保持一致性
    public override bool Equals(object obj)
    {
        return obj is Temperature other && Celsius == other.Celsius;
    }
    
    public override int GetHashCode()
    {
        return Celsius.GetHashCode();
    }
    
    // 重载 == 和 != 运算符以保持完整性
    public static bool operator ==(Temperature t1, Temperature t2)
    {
        if (ReferenceEquals(t1, t2)) return true;
        if (t1 is null || t2 is null) return false;
        
        return t1.Celsius == t2.Celsius;
    }
    
    public static bool operator !=(Temperature t1, Temperature t2)
    {
        return !(t1 == t2);
    }
    
    public override string ToString()
    {
        return $"{Celsius}°C";
    }
}

// 使用示例
Temperature temp1 = new Temperature(25.0);
Temperature temp2 = new Temperature(30.0);
Temperature temp3 = new Temperature(25.0);

Console.WriteLine(temp1 < temp2);   // 输出：True
Console.WriteLine(temp1 > temp2);   // 输出：False
Console.WriteLine(temp1 <= temp3);  // 输出：True
Console.WriteLine(temp1 >= temp3);  // 输出：True

// 使用IComparable接口的方法
Console.WriteLine(temp1.CompareTo(temp2)); // 输出：-1 (表示temp1小于temp2)
Console.WriteLine(temp2.CompareTo(temp1)); // 输出：1  (表示temp2大于temp1)
Console.WriteLine(temp1.CompareTo(temp3)); // 输出：0  (表示相等)

自定义隐式与显式转换

概念：自定义隐式与显式转换

  • 概念：
    • 隐式转换意味着它能够保证转换一定成功，且转换时不会丢失信息。相反，若转换成功与否取决于运行时环境，或转换过程中可能丢失信息，则应当使用显式转换；
    • 而隐式和显式转换也是可重载的运算符，这些转换经过重载后，一般能使强相关的类型（例如数字）之间的转换变得更加简明自然；
    • 但是如果要在弱相关的类型之间进行转换，则更适合采用以下方式：
    • （1）编写一个以转换类型为参数的构造器；
    • （2）编写（静态的）ToXXX 和 FromXXX 方法进行类型转换；
  • 代码：重载 Note 类型和 double 类型之间的转换规则
// 定义 x 转换到 double 的隐式转换
public static implicit operator double(Note x)
{
	return 440 * Math.Pow(2, (double)x.value/2);
}

public static explicit operator Note(double x)
{
	return new Note((int)(0.5 + 12 * (Math.Log(x/440) / Math.Log(2) )));
}

// 使用示例
Note n = (Note)554.37; // 显式转换
double x = n; // 隐式转换
