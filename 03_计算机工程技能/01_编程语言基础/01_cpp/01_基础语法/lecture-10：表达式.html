成员访问表达式

概念：成员访问表达式

  • 概念：点运算符
    • 作用：用于获取类对象的一个成员；
  • 概念：箭头运算符
    • 作用：表示对变量先解引用，然后再获取类对象的一个成员；
      • 即：ptr->mem 等价于 (*ptr).mem；
  • 注意：解引用运算符的优先级低于点运算符
  • 示例：
string s1 = "a string";
string* p = &s1; // 对s1变量取地址
auto n = s1.size();
n = (*p).size(); // 和s1.size()效果一致
n = p->size(); // 和s1.size()效果一致

  • 概念：右移运算符
    • 右移运算符 (>>) 的行为则依赖于其左侧运算对象的类型；
    • 如果该运算对象是无符号类型, 在左侧插入值为 0 的二进制位；
    • 如果该运算对象是带符号类型，在左侧插入符号位的副本或值为 0 的二进制位，如何选择要视具体环境而定；
    • 示例：
  • 分析：移位运算符满足左结合律
    • 举例： cout<<"hi"<<"there"<<endl;
    • 等价于：(((cout<<"hi")<<"there")<<endl;
    • 在这条语句中，运算对象"hi"和第一个 << 组合在一起, 它的结果和第二个 << 组合在一起接下来的结果再和第三个 << 组合在一起；
  • 分析：移位运算符的优先级
    • 移位运算符的优先级不高不低，介于中间；
    • 比算术运算符的优先级低，但比关系运符、赋值运算符和条件运算符的优先级高；
    • 因此，在一次使用多个运算符时，有必要加上括号使其满足我们的要求；

逐位运算

概念：逐位运算

  • 概念：逐位求反
    • int number3 = -1024;cout << ~number3 << endl;
    • 输出：1023
  • 概念：位与、位或、位异或运算符
    • 示例：
    • 与：&，两个都为 1，则结果为 1，否则为 0；
    • 或：|，至少有一个为 1，则为 1，否则为 0；
    • 异或：^，只有一个 1，则结果 1，否则为 0；

sizeof 运算符

概念：sizeof 运算符

  • 概念：
    • sizeof 运算符返回一条表达式或一个类型名字所占的字节数；
    • sizeof 运算符满足右结合律，其所得的值是一个 size_t 类型；
  • 分类：
    • 第一类：sizeof expr
    • 第二类：sizeof(type)
  • 示例：
class Test
{
public:
    int i = 123;
    char j = 'c';
    double k = 123.1234;
};

void operatorTest::Run3()
{
    Test testClass, *ptr;
    cout << "testClass:" << sizeof(testClass) << endl;
    cout << sizeof(ptr) << endl;
    cout << sizeof *ptr << endl;
}
  • 输出：
testClass:16
8
16
  • 分析：
    • 第一个 sizeof (testClass) 表示输出 Test 类型的大小；
    • 第二个 sizeof(ptr) 表示输出指针的大小；
    • 第三个 sizeof *ptr 表示输出 ptr 解引用后的类型的大小，即表示输出 Test 类型的大小；
  • 注意：sizeof expr 中的 expr 不会实际进行运算，因此 expr 的结果可以是空，只要其类型正确即可；
    • 在第二种形式中，sizeof 返回的是表达式结果类型的大小；
    • 与众不同的一点是，sizeof 并不实际计算其运算对象的值，所以即使 ptr 是一个无效 (即未初始化) 的指针也不会有什么影响；
    • 在 sizeof 的运算对象中，解引用一个无效指针仍然是一种安全的行为，因为指针实际上并没有被真正使用；
    • sizeof 不需要真的解引用指针也能知道它所指对象的类型；
