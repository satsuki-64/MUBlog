Lecture 8：函数指针

基础概念

概念：什么是函数指针

  • 概念：
    • 函数指针指向的是函数而非对象；
  • 概念：函数的类型
    • 和其他指针一样，函数指针指向某种特定类型；
    • 函数的类型由它的返回类型和形参类型共同决定，与函数名无关；
    • 比如：bool test(const string&)
    • 其类型为 bool(const string&)；
  • 方法：声明指向函数的指针
    • 使用：
      • 用指针替换函数名即可，比如：
      • bool (*pf)(const string &);
    • 解释：
      • pf 前面有个 *，因此 pf 是指针；
      • 右侧是形参列表，表示 pf 指向的是函数；
      • 再观察左侧，函数的返回类型是布尔值；
      • 因此，p 就是一个指向函数的指针，其中该函数的参数是一个 const string 的引用，返回值是 bool 的类型；
    • 注意：
      • *pf 两端的括号必不可少；
      • 如果不写这对括号，则 pf 是一个返回值为 bool 指针的函数；
  • 方法：使用函数指针
    • 可以将函数名作为一个值使用，此时该函数自动转化为指针；
      • pf = test;
      • 或者是 pf = &test;，这个和 pf = test 等价；
    • 调用指向函数的指针：
      • bool b1 = test("hello");
      • 等价于：bool b1 = pf("hello");

重载函数的指针

概念：重载函数的指针

  • 概念：
    • 当我们使用重载函数时，上下文必须清晰地界定到底应该选用哪个函数；
  • 分析：如果定义了指向重载函数的指针
    • void ff(int*);
    • void ff(unsigned int);
  • 分析：
    • 编译器通过指针类型决定选用哪个函数，指针类型必须与重载函数中的某一个精确匹配；
    • 比如：
      • 正确：void (*pf1)(unsigned int) = ff;
      • 正确：void (*pf2)(int*) = ff;
      • 错误：double (*pf3)(int*) = ff;

函数指针形参

概念：函数指针形参

  • 概念:
    • 虽然不能定义函数类型的形参，但是形参可以是指向函数的指针；
    • 此时，形参看起来是函数类型，实际上却是当成指针使用；
  • 使用：
    • 情况一：形参作为函数类型，自动转化为指向函数的指针
      • void useBigger(const string &s1,const string &s2,bool pf(const string&,const string&));
    • 情况二：显示的将形参定义为指向函数的指针
      • void useBigger(const string &s1,const string &s2,bool (*pf)(const string&,const string&))
    • 使用时：
      • useBigger("Hello","World",testFun);
      • 这里会自动将函数 testFun() 转化为指向该函数的指针；

返回指向函数的指针

概念：返回指向函数的指针

  • 概念：
    • 虽然不能返回一个函数，但是能返回指向函数类型的指针；
    • 然而，我们必须把返回类型写成指针形式，编译器不会自动地将函数返回类型当成对应的指针类型处理；
  • 使用：基于类型别名
    • 要想声明一个返回函数指针的函数，最简单的办法是使用类型别名；
      • using F = int(int*,int);
      • 此时 F 为函数类型，而非指针；
    • 此时可以使用类型别名，将 F 定义为"返回指向函数的指针"的类型：
      • F *f1(int);
