什么是 ASE

ASE 基础概念

概念：ASE 基础概念

  • 概念：
    • Amplify Shader Editor (ASE) 是 Unity 引擎中一款可视化着色器编辑插件，它通过节点图的方式，让开发者能够通过拖拽和连接不同的节点来创建和编辑复杂的着色器，而无需直接编写繁琐的着色器代码；
    • 其自动生成标准的 Unity 着色器代码（HLSL），允许查看和修改生成的源代码，灵活性强；
  • 比较：ASE 与 Shader Graph
    • ​ASE​：问世更早，社区成熟，资源、教程非常丰富。功能强大且灵活，允许用户查看和编辑生成的源代码；
    • Shader Graph​：与引擎的集成度更高，界面和用户体验可能更现代一些。但功能上相对“封装”得更严实，定制灵活性可能不如 ASE；

ASE 的使用

  • 概念：ASE 打开编辑器
    • 打开：
      • 先点击 Show generated code，再点击 Open In Text Editor
    • 然后可以看到完整的代码，并且可以看到其具有两千多个变体；
    • 点进去之后，注意更换其 Shader Type，一般不要使用 Surface：
      • 选择 Legacy 或者 Universal，这两个都是顶点片段着色器；
    • 然后还可以选择 Light Model，即光照模型；
      • 这里选择不收灯光影响的 Unlit；

分析：溶解效果的实现原理

  • 分析：
    • 如果想要物体从原点开始，以溶解的方式、洞一圈之外的地方不显示，只有洞内显示，如何使用？
    • 可以以物体的某个点为圆心，判断周围的顶点到圆心的距离，大于某个距离阈值的被剔除不显示，这样就可以实现溶解；
    • 因此就需要先获得物体上每个顶点的模型本地坐标，然后利用其和圆心的模型本地坐标进行对比；
    • 现在需要，将外面区域给剔除掉，小于半径的才保留；

      • 可以使用 Smoothstep ，其接收距离值，当距离值大于半径时，其值就是 1。小于半径，并大于 FadeRange，在一定范围内有效果；

Shader 中的数组

概念：Shader 中的数组

  • 区别：内置渲染管线与 URP 渲染关系声明数组
  • 使用：内置渲染管线 
    • // 在Properties块中声明，方便在材质面板调整
_MyColorArray ("Color Array", Vector) = (1,1,1,1) // 注意：材质面板无法直接显示数组

// 在CG代码中声明
float4 _MyColorArray[10]; // 声明一个包含10个float4的数组
uniform sampler2D _MyTexArray[4]; // 声明一个包含4张纹理的采样器数组
  • 分析：
    • 虽然在着色器代码中可以声明数组，但 Unity 的材质面板（Properties 块）无法直接为数组元素提供单独的滑块或颜色选择器。通常需要通过 C# 脚本使用 Material.SetVectorArray 等方法来传递数组数据；
  • 使用：URP 
    •   // 必须使用TEXTURE2D和SAMPLER等宏声明纹理和采样器
TEXTURE2D(_MainTex);
SAMPLER(sampler_MainTex);

// 所有材质属性（包括数组）必须放在特定的常量缓冲区(CBUFFER)中
CBUFFER_START(UnityPerMaterial)
    float4 _MainTex_ST; // 纹理缩放偏移
    float4 _MyColorArray[5]; // 颜色数组
CBUFFER_END
  • 代码：C# 脚本向 Shader 传递数据
// 在C#脚本中
public Material targetMaterial;
public Vector4[] colorData;
public Texture2D[] textureData;

void Start()
{
    // 传递Vector数组
    targetMaterial.SetVectorArray("_MyColorArray", colorData);
    // 传递纹理数组（注意：语法为示例，实际传递纹理数组可能需要其他方式）
    // targetMaterial.SetTexture("_MyTexArray", textureData[0]);
}

代码实现

Shader "Unlit/GroundDisappear"
{
    Properties
    {
        _BaseColor("BaseColor",Color) = (1,1,1,1)
        _BaseMap("BaseMap",2D) = "white"{}
        _DisappearPos1("DisappearPos1",Vector) = (0,0,0,0)
        _DisappearPos2("DisappearPos2",Vector) = (0,0,0,0)
        _Radius("Radius",Range(0,10)) = 1
        _Fade("Fade",Range(0,3)) = 1
        _FadeColor("FadeColor",Color) = (1,1,1,1)
    }
    
    SubShader
    {
        Tags
        {
            "RenderType"="Opaque"
            "RenderPipeline" = "UniversalPipeline"
            "IgnoreProjector" = "True"
        }
        LOD 100

        Pass
        {
            Name "SimplesUnlit"
            HLSLPROGRAM
            #pragma vertex vert
            #pragma fragment frag
            #pragma multi_compile_fog
            #include "Packages/com.unity.render-pipelines.core/ShaderLibrary/Color.hlsl"
            #include "Packages/com.unity.render-pipelines.universal/ShaderLibrary/Core.hlsl"
            #include "Packages/com.unity.render-pipelines.universal/ShaderLibrary/Lighting.hlsl"

            struct Attributes
            {
                float4 positionOS : POSITION;
                float2 uv : TEXCOORD0;
            };

            struct Varyings
            {
                float4 positionCS : SV_POSITION;
                float2 uv : TEXCOORD0;
                float fogCoord : TEXCOORD1;
                float3 positionWS : TEXCOORD2;
            };

			// 变量定义在 CBuffer 当中
            CBUFFER_START(UnityPerMaterial)
                half4 _BaseColor;
                half4 _DisappearPos1;
                half4 _DisappearPos2;
                half _Radius;
                half _Fade;
                half4 _FadeColor;
            CBUFFER_END
            
            TEXTURE2D(_BaseMap);SAMPLER(sampler_BaseMap);float4 _BaseMap_ST;

            Varyings vert (Attributes v)
            {
                Varyings o;
                o.positionWS = TransformObjectToWorld(v.positionOS);
                o.positionCS = TransformObjectToHClip(v.positionOS);
                o.uv = TRANSFORM_TEX(v.uv,_BaseMap);
                o.fogCoord = ComputeFogFactor(o.positionCS.z);

                return o;
            }

            half4 frag (Varyings i) : SV_Target
            {
                float distance0 = distance(_DisappearPos1,i.positionWS);
                float fadeRange = _Radius - _Fade;
                
                // 将半径以外的区域剔除掉
                clip(_Radius-distance0);
                // 在 FadeRange 区域内，叠加光滑过渡的颜色
                
                half4 color;
                half4 baseMap = SAMPLE_TEXTURE2D(_BaseMap,sampler_BaseMap,i.uv);
                color = baseMap * _BaseColor;

                float fadeInstensity = (saturate(distance0 - fadeRange));
                color += (_FadeColor * fadeInstensity);
                color.rgb = MixFog(color.rgb,i.fogCoord);
                
                return color;
            }
            
            ENDHLSL
        }
    }
}
