水体基础效果实现

水体效果分析

概念：水体效果分析

  • 概念：
    • 水体效果主要分为几个部分：
      1. 水的深度：
      1. 水的高光；
      1. 水的反射；
      1. 水的焦散效果；
      1. 水下的扭区；
      1. 水面的泡沫；
  • 概念：水的深度
    • 效果：
      • 比如水面往下的对象观察的效果会和深度有关，越是靠近水面的物体看的越清楚、颜色越清澈，越是远的物体看的越模糊、颜色越重；
      • 水的深度是水体渲染的基石，它直接影响颜色的过渡、透明度以及许多后续效果（如泡沫）的生成；
    • 原理：
      • 深度计算的核心目标是获取水面某一点到水底物体的垂直距离。在 URP 中，这通常通过对比两种深度值来实现；
          1. 场景深度​：从摄像机到水底物体表面的距离，通过深度纹理获取；
          1. 水面深度​：从摄像机到水面模型本身的距离；
      • 两者相减，即可得到我们需要的垂直水深；
    • 实现：
        1. 启用纹理​：在 URP 资源文件中，启用 Depth Texture 和 Opaque Texture；
        1. 计算垂直水深；
        • 注意，水体本身需要时半透明，不写入深度；
        1. 颜色插值；

水的深度

概念：水的深度

  • 概念：
    • 主要分为两步，分别是获得场景中物体的深度（通过深度图采样），以及获得当前水面物体在视图空间下的深度 Z，然后由于
  • 实现：
half depthTex = SAMPLE_TEXTURE2D(_CameraDepthTexture, sampler_CameraDepthTexture, screenUV);
half depthScene = LinearEyeDepth(depthTex, _ZBufferParams);
half depthWater = depthScene + i.positonVS.z;
  • 解释：
    • depthScene ​：从相机到场景物体表面的直线距离（观察空间下的正值）
    • i.positonVS.z​：从相机到水面表面的直线距离（观察空间下的负值）
    • 而在 Unity 的观察空间（右手坐标系）中，相机位于原点 (0,0,0)，Z 轴指向相机后方，所以相机前方的物体 Z 坐标为负值，因此假设：
    • 因此可以计算得到的水深为 5 个单位，这正好是水面到水底物体的垂直距离；

水体泡沫效果

概念：水体泡沫效果

  • 概念：
    • 水体泡沫效果相对而言比较独立，只需要水的深度；
    • 其中，foamRange 表示泡沫存在的范围，foamTex 从噪声图采样、得到多样的泡沫效果，foamMask 用于只计算泡沫区域的颜色，将 foamMask * _FoamColor 得到泡沫区域的颜色，然后再将泡沫颜色和水体颜色相加，得到完整的颜色；
  • 代码：
Shader "Unlit/Water"
{
    Properties
    {
        _Speed("Speed",Range(0,0.5)) = 0.1
        
        _WaterColor01("WaterColor01",Color) = (1,1,1,1)
        _WaterColor02("WaterColor02",Color) = (1,1,1,1)
        
        [Header(Foam)]
        _FoamTex("FoamTex",2D) = "white"{}
        _FoamRange("FoamRange",Range(0,3)) = 1
        _FoamColor("FoamColor",Color) = (1,1,1,1)
        _FoamNoise("FoamNoise",Range(0,1)) = 0.5
        
        _Cull("__cull", Float) = 2.0
        _Cutoff("Alpha Clipping", Range(0.0, 1.0)) = 0.5
    }
    
    SubShader
    {
        Tags
        {
            "RenderType" = "Transparent"
            "Queue" = "Transparent"
            "RenderPipeline" = "UniversalPipeline"
            "IgnoreProjector" = "True"
        }
        LOD 100
        ZWrite On
        ZTest LEqual
        Cull[_Cull]

        Pass
        {
            Name "Water"
            HLSLPROGRAM
            #pragma vertex vert
            #pragma fragment frag
            #pragma multi_compile_fog
            #include "Packages/com.unity.render-pipelines.core/ShaderLibrary/Color.hlsl"
            #include "Packages/com.unity.render-pipelines.universal/ShaderLibrary/Core.hlsl"
            #include "Packages/com.unity.render-pipelines.universal/ShaderLibrary/Lighting.hlsl"
            #include "Packages/com.unity.render-pipelines.universal/ShaderLibrary/ShaderGraphFunctions.hlsl"

            
            struct Attributes
            {
                float4 positionOS : POSITION;
                float2 uv : TEXCOORD0;
            };

            struct Varyings
            {
                float4 positionCS : SV_POSITION;
                float4 uv : TEXCOORD0; // xy：foamUV
                float fogCoord : TEXCOORD1;
                float3 positonVS : TEXCOORD2;
                float3 positionWS : TEXCOORD3;
            };

            CBUFFER_START(UnityPerMaterial)
                half4 _WaterColor01;
                half4 _WaterColor02;
                half _FoamRange;
                half4 _FoamColor;
                half _Speed;
                half _FoamNoise;
            CBUFFER_END

            TEXTURE2D(_CameraDepthTexture);SAMPLER(sampler_CameraDepthTexture);
            TEXTURE2D(_FoamTex);SAMPLER(sampler_FoamTex);float4 _FoamTex_ST;

            Varyings vert (Attributes v)
            {
                Varyings o;

                // 将顶点从本地坐标转换为世界坐标，然后再转换为观察空间
                float3 positionWS = TransformObjectToWorld(v.positionOS);
                o.positionWS = positionWS;
                o.positonVS = TransformWorldToView(positionWS);
                o.positionCS = TransformObjectToHClip(v.positionOS);
                
                o.uv.xy = o.positionWS.xz * _FoamTex_ST.xy + _Time.y * _Speed;
                
                o.fogCoord = ComputeFogFactor(o.positionCS.z);

                return o;
            }

            half4 frag (Varyings i) : SV_Target
            {
                // 屏幕空间下的 UV 坐标
                float2 screenUV = i.positionCS.xy / _ScreenParams.xy;
                
                // --------------------- 水的深度 ------------------------
                // 1. 获取场景中物体的深度
                half depthTex = SAMPLE_TEXTURE2D(_CameraDepthTexture,sampler_CameraDepthTexture,screenUV);
                half depthScene = LinearEyeDepth(depthTex,_ZBufferParams); // 物体的转换到观察空间下的Z值
                
                // 2. 计算物体本身的观察空间下的Z值，即物体在水下的深度值
                half depthWater = depthScene + i.positonVS.z;

                // --------------------- 水的颜色 ------------------------
                half4 waterColor = lerp(_WaterColor01,_WaterColor02,depthWater);
                
                
                // --------------------- 水的泡沫 ------------------------
                half foamRange = depthWater* _FoamRange;
                half foamTex = SAMPLE_TEXTURE2D(_FoamTex,sampler_FoamTex,i.uv.xy);
                foamTex *= pow(foamTex,_FoamNoise);
                half foamMask = step(foamRange,foamTex);
                half4 foam = foamMask * _FoamColor;
                
                return waterColor + foam;
            }
            
            ENDHLSL
        }
    }
}

水下扭区效果

分析：水下扭区效果

  • 分析：混合模式
    • 水面首先需要能够看到水下的物体、让其透过水面，然后再看到扭区的效果；
    • 因此首先加入 Blend SrcAlpha OneMinusSrcAlpha，然后再继续分析；
  • 分析：水下扭区实现
      1. 首先分析如何实现扭区效果：首先准备一张噪声图，然后使用这个噪声图的采样值、来作为采样主贴图的 UV，就可以实现扭区的效果；
      1. 如果想要实现水下的扭区，只需要先进行抓屏，然后再对抓屏后的画面进行采样，采样的 UV 使用屏幕空间 UV 和噪声图的采样值的 lerp 插值，即可实现不同程度的扭区；
  • 分析：完整实现分析

代码：核心代码实现

// --------------------- 水的扭区 ------------------------
half4 distortValue = SAMPLE_TEXTURE2D(_DistortTex,sampler_DistortTex,i.uv.zw);
float2 distortUV = lerp(screenUV,distortValue,_Distort);
float2 opaqueUV;
// 由于只使用抓屏、无法区分哪些地方该扭区、哪些地方不该扭曲，因此需要利用上当前的深度值信息，通过深度值、计算出来哪些部分在水面以下
half depthDistortTex = SAMPLE_TEXTURE2D(_CameraDepthTexture,sampler_CameraDepthTexture,distortUV);
half depthDistortScene = LinearEyeDepth(depthDistortTex,_ZBufferParams);
half depthDistortWater = depthDistortScene + i.positonVS.z;
// 如果 depthDistortWater < 0 则代表当前面片对应的物体的“水深”小于0，代表其位于水面之上，因此其不进行扭区
if (depthDistortWater < 0)
{
	opaqueUV = screenUV;
}
else
{
	opaqueUV = distortUV;
}
half4 opaqueTex = SAMPLE_TEXTURE2D(_CameraOpaqueTexture,sampler_CameraOpaqueTexture,opaqueUV);

Color = opaqueTex * waterColor + foam;
Color.a = 0.5;
return Color;

      • 采样一张法线凹凸贴图，并且分别对其 U 和 V 方向进行流动，然后分别采样 xw、zw，将它们相乘，使得法线贴图在 U 和 V 方向上流动，然后再使用这个凹凸贴图的采样值作为光照计算的法线；

Shader "Unlit/Water"
{
    Properties
    {
        [Header(Base)]
        _Speed("Speed",Range(0,0.1)) = 0.1
        _Alpha("Alpha",Range(0,1)) = 0.8
        
        [Header(Color)]
        _WaterColor01("WaterColor01",Color) = (1,1,1,1)
        _WaterColor02("WaterColor02",Color) = (1,1,1,1)
        _SpecularInstensity("SpecularInstensity",Float) = 1
        _SpecularSmoothness("SpecularSmoothness",Float) = 1
        _NormalTex("NormalTex",2D) = "white"{}
        
        [Header(Foam)]
        _FoamTex("FoamTex",2D) = "white"{}
        _FoamRange("FoamRange",Range(0,3)) = 1
        _FoamColor("FoamColor",Color) = (1,1,1,1)
        _FoamNoise("FoamNoise",Range(0,1)) = 0.5
        
        [Header(Distort)]
        _Distort("Distort",Range(0,1)) = 0.3
        _DistortTex("DistortTex",2D) = "white"{}
    }
    
    SubShader
    {
        Tags
        {
            "RenderType" = "Transparent"
            "Queue" = "Transparent"
            "RenderPipeline" = "UniversalPipeline"
            "IgnoreProjector" = "True"
        }
        LOD 100
        ZWrite On
        ZTest LEqual
        Blend SrcAlpha OneMinusSrcAlpha

        Pass
        {
            Name "Water"
            HLSLPROGRAM
            #pragma vertex vert
            #pragma fragment frag
            #pragma multi_compile_fog
            #include "Packages/com.unity.render-pipelines.core/ShaderLibrary/Color.hlsl"
            #include "Packages/com.unity.render-pipelines.universal/ShaderLibrary/Core.hlsl"
            #include "Packages/com.unity.render-pipelines.universal/ShaderLibrary/Lighting.hlsl"
            #include "Packages/com.unity.render-pipelines.universal/ShaderLibrary/ShaderGraphFunctions.hlsl"

            
            struct Attributes
            {
                float4 positionOS : POSITION;
                float2 uv : TEXCOORD0;
                float3 normalOS : NORMAL;
            };

            struct Varyings
            {
                float4 positionCS : SV_POSITION;
                float4 uv : TEXCOORD0; // xy：foamUV
                float fogCoord : TEXCOORD1;
                float3 positonVS : TEXCOORD2;
                float3 positionWS : TEXCOORD3;
                float3 normalWS : TEXCOORD4;
                float4 normalUV : TEXCOORD5;
            };

            CBUFFER_START(UnityPerMaterial)
                half _Alpha;
                half4 _WaterColor01;
                half4 _WaterColor02;
                half4 _SpecularColor;
                half _SpecularInstensity;
                half _SpecularSmoothness;
                half _FoamRange;
                half4 _FoamColor;
                half _Speed;
                half _FoamNoise;
                half _Distort;
            CBUFFER_END

            TEXTURE2D(_CameraDepthTexture);SAMPLER(sampler_CameraDepthTexture);
            TEXTURE2D(_FoamTex);SAMPLER(sampler_FoamTex);float4 _FoamTex_ST;
            TEXTURE2D(_CameraOpaqueTexture);SAMPLER(sampler_CameraOpaqueTexture);
            TEXTURE2D(_DistortTex);SAMPLER(sampler_DistortTex);float4 _DistortTex_ST;
            TEXTURE2D(_NormalTex);SAMPLER(sampler_NormalTex);float4 _NormalTex_ST;

            Varyings vert (Attributes v)
            {
                Varyings o;

                // 将顶点从本地坐标转换为世界坐标，然后再转换为观察空间
                float3 positionWS = TransformObjectToWorld(v.positionOS);
                o.positionWS = positionWS;
                o.positonVS = TransformWorldToView(positionWS);
                o.positionCS = TransformObjectToHClip(v.positionOS);

                // UV 采样
                float speed = _Time.y * _Speed;
                o.uv.xy = o.positionWS.xz * _FoamTex_ST.xy + speed;
                o.uv.zw = TRANSFORM_TEX(v.uv,_FoamTex) + speed;
                o.normalUV.xy = TRANSFORM_TEX(v.uv,_NormalTex) + speed * float2(1,1); 
                o.normalUV.zw = TRANSFORM_TEX(v.uv,_NormalTex) + speed * float2(-1,1);
                o.uv.zw = TRANSFORM_TEX(v.uv,_DistortTex) + speed;
                
                // 法线与雾效
                o.normalWS = TransformObjectToWorldNormal(v.normalOS);
                o.fogCoord = ComputeFogFactor(o.positionCS.z);

                return o;
            }

            half4 frag (Varyings i) : SV_Target
            {
                float4 Color;
                
                // 屏幕空间下的 UV 坐标
                float2 screenUV = i.positionCS.xy / _ScreenParams.xy;
                
                // --------------------- 水的深度 ------------------------
                // 1. 获取场景中物体的深度
                half depthTex = SAMPLE_TEXTURE2D(_CameraDepthTexture,sampler_CameraDepthTexture,screenUV);
                half depthScene = LinearEyeDepth(depthTex,_ZBufferParams); // 物体的转换到观察空间下的Z值
                 
                // 2. 计算物体本身的观察空间下的Z值，即物体在水下的深度值
                half depthWater = depthScene + i.positonVS.z;

                // --------------------- 水的颜色 ------------------------
                half4 waterColor = lerp(_WaterColor01,_WaterColor02,depthWater);

                // --------------------- 水的高光 ------------------------
                half4 normalTex01 = SAMPLE_TEXTURE2D(_NormalTex,sampler_NormalTex,i.normalUV.xy);
                half4 normalTex02 = SAMPLE_TEXTURE2D(_NormalTex,sampler_NormalTex,i.normalUV.zw);
                half4 normalTex = normalTex01 * normalTex02;
                half4 N = normalTex;
                Light light = GetMainLight();
                half4 lightColor = float4(light.color,0.5);
                // 光照的方向
                half3 L = light.direction;
                // 顶点指向摄像机的方向
                half3 V = normalize(_WorldSpaceCameraPos.xyz - i.positionWS.xyz);
                half3 H = normalize(L + V);
                half NdotH = dot(N,H);
                half4 specular = lightColor * _SpecularInstensity * pow(saturate(NdotH),_SpecularSmoothness);
                
                // --------------------- 水的泡沫 ------------------------
                half foamRange = depthWater* _FoamRange;
                half foamTex = SAMPLE_TEXTURE2D(_FoamTex,sampler_FoamTex,i.uv.xy);
                foamTex *= pow(foamTex,_FoamNoise);
                half foamMask = step(foamRange,foamTex);
                half4 foam = foamMask * _FoamColor;
                
                // --------------------- 水的扭区 ------------------------
                half4 distortValue = SAMPLE_TEXTURE2D(_DistortTex,sampler_DistortTex,i.uv.zw);
                float2 distortUV = lerp(screenUV,distortValue,_Distort);
                float2 opaqueUV;
                // 由于只使用抓屏、无法区分哪些地方该扭区、哪些地方不该扭曲，因此需要利用上当前的深度值信息，通过深度值、计算出来哪些部分在水面以下
                half depthDistortTex = SAMPLE_TEXTURE2D(_CameraDepthTexture,sampler_CameraDepthTexture,distortUV);
                half depthDistortScene = LinearEyeDepth(depthDistortTex,_ZBufferParams);
                half depthDistortWater = depthDistortScene + i.positonVS.z;
                // 如果 depthDistortWater < 0 则代表当前面片对应的物体的“水深”小于0，代表其位于水面之上，因此其不进行扭区
                if (depthDistortWater < 0)
                {
                    opaqueUV = screenUV;
                }
                else
                {
                    opaqueUV = distortUV;
                }
                half4 opaqueTex = SAMPLE_TEXTURE2D(_CameraOpaqueTexture,sampler_CameraOpaqueTexture,opaqueUV);

                Color = opaqueTex * waterColor + foam + specular;
                Color.a = _Alpha;
                return Color;
            }
            
            ENDHLSL
        }
    }

水的反射效果

反射探针与环境贴图

概念：反射探针与环境贴图

  • 概念：反射探针
    • 反射探针是 Unity 中用于模拟环境反射的核心组件，它通过在场景中特定位置捕获周围环境的六个方向视图，生成立方体贴图（Cubemap），然后应用于具有反射材质的物体表面；
    • 在 Hierarchy 面板中右键点击，选择 ​ Light > Reflection Probe，或者通过菜单 ​ GameObject > Light > Reflection Probe ​ 来创建；
    • 将反射探针放置在需要反射的关键位置，如房间中心、窗户附近等，调整 ​Box Size​ 属性，使黄色方框覆盖需要反射的区域，使用 ​Probe Origin​ 调整采样点的位置，通常放置在窗户或反射表面附近；
  • 概念：反射探针相关属性
    • unity_SpecCube0：类型为 TextureCube，包含反射探针烘焙或实时捕获的环境立方体贴图（Cubemap）
    • samplerunity_SpecCube0：类型为 SamplerState，与 unity_SpecCube0 关联的采样器状态；
    • unity_SpecCube0_HDR：类型为 float4，存储 HDR 解码参数，如贴图的缩放和偏移，用于正确解码颜色值；
    • SAMPLE_TEXTURECUBE：用于对立方体贴图进行采样，返回未经解码的 RGBM 等编码格式的颜色值；
    • DecodeHDREnvironment：对 SAMPLE_TEXTURECUBE 采样的结果进行解码，还原为 HDR 颜色值；

分析：水的反射 - 第一步：采样反射贴图

  • 分析：
    • 水的反射不会使用实时计算的方式，这样的性能开销太大。这里使用一张 Cubemap 来记录环境；
    • 反射图是将场景烘培成一张反射贴图；
  • 概念：Cubemap 及其采样
    • Cubemap 是由 6 个独立的 2D 纹理组成的立方体，每个面代表一个方向（±X、±Y、±Z）
    • 采样的基本原理是：从原点发出一个方向向量，该向量会与立方体的某个面相交，交点处的纹理就是采样结果；
  • 公式：
    • 在 Shader 当中，使用计算得到的反射向量来对 Cubemap 进行采样；
  • 实现：在 Shader 当中，使用 reflect 函数计算反射向量 
    •   float3 viewDir = normalize(_WorldSpaceCameraPos.xyz - i.pos_world);
  // 传入视线方向的反方向，以及法线方向
float3 reflectDir = reflect(-viewDir, normal_dir);
fixed4 col = texCUBE(_CubeMap, reflectDir);

	_Speed("Speed",Range(0,0.1)) = 0.1
	_Alpha("Alpha",Range(0,1)) = 0.8
	
	[Header(Color)]

	_WaterColor02("WaterColor02",Color) = (1,1,1,1)

	_SpecularInstensity("SpecularInstensity",Float) = 1
	_SpecularSmoothness("SpecularSmoothness",Float) = 1

	_CausticScale("CausticScale",Range(0,1)) = 0.5

SubShader
{
	Tags
	{
		"RenderType" = "Transparent"
		"Queue" = "Transparent"
		"RenderPipeline" = "UniversalPipeline"
		"IgnoreProjector" = "True"
	}
	LOD 100
	ZWrite On
	ZTest LEqual
	Blend SrcAlpha OneMinusSrcAlpha

	Pass
	{
		Name "Water"
		HLSLPROGRAM
		#pragma vertex vert
		#pragma fragment frag
		#pragma multi_compile_fog
		#include "Packages/com.unity.render-pipelines.core/ShaderLibrary/Color.hlsl"
		#include "Packages/com.unity.render-pipelines.universal/ShaderLibrary/Core.hlsl"
		#include "Packages/com.unity.render-pipelines.universal/ShaderLibrary/Lighting.hlsl"

		
		struct Attributes
		{
			float4 positionOS : POSITION;
			float2 uv : TEXCOORD0;
			float3 normalOS : NORMAL;
		};

		struct Varyings
		{
			float4 positionCS : SV_POSITION;
			float4 uv : TEXCOORD0; // xy：foamUV
			float fogCoord : TEXCOORD1;
			float3 positonVS : TEXCOORD2;
			float3 positionWS : TEXCOORD3;
			float3 normalWS : TEXCOORD4;
			float4 normalUV : TEXCOORD5;
		};

		CBUFFER_START(UnityPerMaterial)
			half _Alpha;
			half4 _WaterColor01;
			half4 _WaterColor02;
			half _WaterDepth;
			half4 _SpecularColor;
			half _SpecularInstensity;
			half _SpecularSmoothness;
			half _FoamRange;
			half4 _FoamColor;
			half _Speed;
			half _FoamNoise;
			half _Distort;
			half _NormalInstensity;
			half _ReflectionInstensity;
			half _CausticScale;
			half _CausticScaleY;
			half _CausticInstensity;
		CBUFFER_END

		TEXTURE2D(_CameraDepthTexture);SAMPLER(sampler_CameraDepthTexture);
		TEXTURE2D(_FoamTex);SAMPLER(sampler_FoamTex);float4 _FoamTex_ST;
		TEXTURE2D(_CameraOpaqueTexture);SAMPLER(sampler_CameraOpaqueTexture);
		TEXTURE2D(_DistortTex);SAMPLER(sampler_DistortTex);float4 _DistortTex_ST;
		TEXTURE2D(_NormalTex);SAMPLER(sampler_NormalTex);float4 _NormalTex_ST;
		TEXTURE2D(_CausticTex);SAMPLER(sampler_CausticTex);
		TEXTURECUBE(_ReflectionTex);SAMPLER(sampler_ReflectionTex);

		Varyings vert (Attributes v)
		{
			Varyings o;

			// 将顶点从本地坐标转换为世界坐标，然后再转换为观察空间
			float3 positionWS = TransformObjectToWorld(v.positionOS.xyz);
			o.positionWS = positionWS;
			o.positonVS = TransformWorldToView(positionWS);
			o.positionCS = TransformObjectToHClip(v.positionOS.xyz);

			// UV 采样
			float speed = _Time.y * _Speed;
			o.uv.xy = o.positionWS.xz * _FoamTex_ST.xy + speed;
			o.uv.zw = TRANSFORM_TEX(v.uv,_FoamTex) + speed;
			o.normalUV.xy = TRANSFORM_TEX(v.uv,_NormalTex) + speed * float2(1,1); 
			o.normalUV.zw = TRANSFORM_TEX(v.uv,_NormalTex) + speed * float2(-1.07,1.3);
			o.uv.zw = TRANSFORM_TEX(v.uv,_DistortTex) + speed;
			
			// 法线与雾效
			o.normalWS = TransformObjectToWorldNormal(v.normalOS);
			o.fogCoord = ComputeFogFactor(o.positionCS.z);

			return o;
		}

		half4 frag (Varyings i) : SV_Target
		{
			float4 Color;
			
			// 屏幕空间下的 UV 坐标
			float2 screenUV = i.positionCS.xy / _ScreenParams.xy;
			
			// --------------------- 水的深度 ------------------------
			// 1. 获取场景中物体的深度
			half depthTex = SAMPLE_TEXTURE2D(_CameraDepthTexture,sampler_CameraDepthTexture,screenUV).r;
			half depthScene = LinearEyeDepth(depthTex,_ZBufferParams); // 物体的转换到观察空间下的Z值
			 
			// 2. 计算物体本身的观察空间下的Z值，即物体在水下的深度值
			half depthWater = depthScene + i.positonVS.z;
			depthWater *= _WaterDepth;

			// --------------------- 水的颜色 ------------------------
			half4 waterColor = lerp(_WaterColor01,_WaterColor02,depthWater);

			// --------------------- 水的高光 ------------------------
			// 1. 采样法线图，使得水面看起来凹凸起伏
			half4 normalTex01 = SAMPLE_TEXTURE2D(_NormalTex,sampler_NormalTex,i.normalUV.xy);
			half4 normalTex02 = SAMPLE_TEXTURE2D(_NormalTex,sampler_NormalTex,i.normalUV.zw);
			half4 normalTex = normalTex01 * normalTex02;
			// 2. 使用 Blinne-Phone 模型计算高光
			half3 N = lerp(normalize(i.normalWS),normalTex.xyz,_NormalInstensity);
			Light light = GetMainLight();
			half4 lightColor = float4(light.color,0.5);
			half3 L = light.direction; // 光照的方向
			half3 V = normalize(_WorldSpaceCameraPos.xyz - i.positionWS.xyz); // 顶点指向摄像机的方向
			half3 H = normalize(L + V);
			half NdotH = dot(N,H);
			// 3. 计算高光颜色
			half4 specular = lightColor * _SpecularInstensity * pow(saturate(NdotH),_SpecularSmoothness);

			// --------------------- 水的反射 ------------------------
			half3 reflectionUV = reflect(V,N);
			half4 encodedColor = SAMPLE_TEXTURECUBE(_ReflectionTex,sampler_ReflectionTex,reflectionUV);
			half fresnel = pow(1-saturate(dot(N,V)),_ReflectionInstensity);
			half4 reflection = encodedColor * fresnel;
			
			// --------------------- 水的泡沫 ------------------------
			half foamRange = depthWater* _FoamRange;
			half foamTex = SAMPLE_TEXTURE2D(_FoamTex,sampler_FoamTex,i.uv.xy).r;
			foamTex *= pow(abs(foamTex),_FoamNoise);
			half foamMask = step(foamRange,foamTex);
			half4 foam = foamMask * _FoamColor;

			
			// --------------------- 水的扭区 ------------------------
			// 1. 计算扭区 UV，_Distort=0 时使用屏幕空间 UV
			half4 distortValue = SAMPLE_TEXTURE2D(_DistortTex,sampler_DistortTex,i.uv.zw);
			float2 distortUV = screenUV + distortValue.xy*_Distort;
			float2 opaqueUV;
			// 2. 由于只使用抓屏、无法区分哪些地方该扭区、哪些地方不该扭曲，因此需要利用上当前的深度值信息，通过深度值、计算出来哪些部分在水面以下
			half depthDistortTex = SAMPLE_TEXTURE2D(_CameraDepthTexture,sampler_CameraDepthTexture,distortUV).r;
			half depthDistortScene = LinearEyeDepth(depthDistortTex,_ZBufferParams);
			half depthDistortWater = depthDistortScene + i.positonVS.z;
			// 3. 如果 depthDistortWater < 0 则代表当前面片对应的物体的“水深”小于0，代表其位于水面之上，因此其不进行扭区
			if (depthDistortWater < 0)
			{
				opaqueUV = screenUV;
			}
			else
			{
				opaqueUV = distortUV;
			}
			half4 opaqueTex = SAMPLE_TEXTURE2D(_CameraOpaqueTexture,sampler_CameraOpaqueTexture,opaqueUV);
			
			// --------------------- 水的焦散 ------------------------
			// 1. 观察空间下的深度点
			float4 depthVS = 1;
			depthVS.xy = i.positonVS.xy * depthDistortScene / - i.positonVS.z;
			depthVS.z = depthDistortScene;
			float3 depthWS = mul(unity_CameraToWorld,depthVS).xyz;
			// 2. 计算U和V方向上流动的UV
			float2 causticUV01 = depthWS.xz * _CausticScale + depthWS.y * _CausticScaleY + _Time.y * _Speed;
			float2 causticUV02 = depthWS.xz * _CausticScale * 0.86 + depthWS.y * _CausticScaleY + _Time.y * _Speed * float2(-1.62,1.21);
			// 3. 计算焦散颜色，并使用 min 增强焦散的偏移效果
			half4 causticTex01 = SAMPLE_TEXTURE2D(_CausticTex,sampler_CausticTex,causticUV01);
			half4 causticTex02 = SAMPLE_TEXTURE2D(_CausticTex,sampler_CausticTex,causticUV02);
			half4 caustic = min(causticTex01,causticTex02) * _CausticInstensity;
			
			// --------------------- 效果叠加 ------------------------
			Color = opaqueTex * waterColor;
			Color += caustic;
			Color += specular;
			Color += reflection;
			Color += foam;
			Color.a = _Alpha;
			return Color;
		}
		
		ENDHLSL
	}
}

前置知识：Texture2D 类

Texture2D 类的创建

概念：Texture2D 类

  • 概念：
    • Texture2D 是 Unity 中处理二维纹理的核心类，广泛应用于 Spite、UI、材质贴图等场景；
  • 概念：Texture2D 类创建参数
    • （1）纹理格式：
      • 纹理格式定义了每个像素在内存中的存储方式，直接影响图像质量、大小和兼容性；
        1. RGBA32：每个像素 8 位，包含红、绿、蓝和透明度通道，是最常用的格式之一；
        1. RGB24：每个像素 8 位，仅包含红、绿、蓝通道，​无透明度；
        1. ARGB32 ​：类似 RGBA32，但 Alpha（透明度）通道顺序不同；
        1. 其他格式​：如 RGBAFloat（高精度 HDR）、BC（压缩格式，节省内存但可能有损）
    • （2）MipMap：
      • 是一系列预先计算、逐级缩小的纹理链。当三维物体远离相机时，GPU 会自动切换到更小的 MipMap 级别进行采样，这有助于减少锯齿​（闪烁的像素点），提升视觉质量，尤其适用于 3D 场景中的物体；
      • 启用 MipMap 会增加约三分之一的内存占用；
    • （3）线性空间：
      • 线性空间 (Linear)​​ 此参数控制纹理的采样方式；
        1. 若设为 true，纹理将在线性颜色空间中被采样，适用于需要物理精确光照的计算（如法线贴图、金属度贴图等）
        1. 若设为 false，则在 ​ Gamma 颜色空间​（sRGB）中被采样，适用于普通颜色贴图；
      • 在 Unity 的 Project Settings -> Player 中启用线性空间渲染时，此设置尤为重要；
  • 概念：Texture2D 创建方式
// 创建一个 512x512 的空白纹理，格式为ARGB32，不生成MipMap
Texture2D texture = new Texture2D(512, 512);

// 创建一个 256x256 的纹理，格式为RGBA32，生成MipMap，使用线性滤波
Texture2D textureWithMip = new Texture2D(256, 256, TextureFormat.RGBA32, true, true);

// 创建一个 128x128 的纹理，格式为RGB24，不生成MipMap，使用sRGB采样（false）
Texture2D textureRGB = new Texture2D(128, 128, TextureFormat.RGB24, false, false);
  • 概念：Texture2D 从本地加载
// 方法一：使用 LoadImage (适用于解码PNG/JPG等格式)
public Texture2D LoadTextureFromFile(byte[] fileData) 
{
	// 初始尺寸会被图片实际尺寸覆盖
    Texture2D texture = new Texture2D(512, 512);
     
    // LoadImage 会根据文件数据自动重置纹理尺寸和格式
    if (texture.LoadImage(fileData)) 
    { 
        return texture;
    }
    
    return null;
}

// 方法二：使用 LoadRawTextureData (适用于已知格式的原始数据)
public Texture2D LoadTextureFromRawData(byte[] rawData, int width, int height, TextureFormat format) 
{
    Texture2D texture = new Texture2D(width, height, format, false);
    texture.LoadRawTextureData(rawData);
    
    // 必须调用Apply使数据生效
    texture.Apply(); 
    
    return texture;
}
  • 概念：从现有 Texture 转换
// 如果需要修改从资源加载的纹理，或者处理 RenderTexture的内容，可以使用以下方法转换
public Texture2D ConvertRenderTextureToTexture2D(RenderTexture renderTex) 
{
    // 保存当前激活的 RenderTexture
    RenderTexture previous = RenderTexture.active;
    
    // 设置目标 RenderTexture 为当前激活状态
    RenderTexture.active = renderTex;

    // 创建一个新的 Texture2D 来存储像素数据
    Texture2D tex2D = new Texture2D(renderTex.width, renderTex.height, TextureFormat.RGBA32, false);
    
    // 从当前激活的 RenderTexture 中读取像素
    tex2D.ReadPixels(new Rect(0, 0, renderTex.width, renderTex.height), 0, 0);
    tex2D.Apply();

    // 恢复之前激活的 RenderTexture
    RenderTexture.active = previous;
    return tex2D;
}

Texture2D 类常用方法

概念：Texture2D 类常用方法

  • 概念：像素操作 - SetPixel、GetPixel 与 Apply
    • （1）void SetPixel(int x, int y, Color color)​
      • 设置纹理上特定坐标（x, y）处的像素颜色；
      • x：X 坐标（0 到 width-1），y 同理；
      • SetPixel 调用后必须调用 Apply 方法才能使更改生效。频繁调用 SetPixel 性能较低，如需设置大量像素，应使用 SetPixels；
    • （2）Color GetPixel(int x, int y)
      • 获取纹理上特定坐标的像素颜色，此方法需要纹理的 isReadable 属性为 true（在纹理导入设置中启用 Read/Write Enabled）
    • （3）void Apply(bool updateMipmaps = true, bool makeNoLongerReadable = false)
      • 将所有通过 SetPixel 或 SetPixels 进行的更改上传到显卡并实际应用到纹理；
  • 概念：批量像素操作 - SetPixels 与 GetPixels
    • （1）void SetPixels(Color[] colors)​
      • 用 Color 数组一次性设置整个纹理或指定矩形的像素；
      • 其他重载：SetPixels(int x, int y, int blockWidth, int blockHeight, Color[] colors, int miplevel = 0)
  • 概念：编码与保存 - EncodeToPNG 与 EncodeToJPG
    • byte[] EncodeToPNG()：将纹理编码为 PNG 格式的字节数组。​ PNG 格式支持无损压缩和透明度；
  • 示例：
public void SaveTextureAsPNG(Texture2D texture, string filePath) 
{
    byte[] pngData = texture.EncodeToPNG();
    System.IO.File.WriteAllBytes(filePath, pngData);
    Debug.Log("Texture saved to: " + filePath);
}

C# 脚本动态生成渐变图

public class WaterController : MonoBehaviour
{
	/// <summary>
	/// 水体的渐变纹理
	/// </summary>
	[Header("水体的渐变纹理")]
	public Texture2D RampTexture;
	
	[Header("水体渐变效果控制")]
	public Gradient waterGradient;
	
	public int count;
	
	private void Start()
	{
		// 参数一、二：材质的宽和高，单位为像素
		RampTexture = new Texture2D(512, 256);
		RampTexture.wrapMode = TextureWrapMode.Clamp;
		
		count = RampTexture.width * RampTexture.height;
		Color[] colors = new Color[count];
		for (int i = 0; i < count; i++)
		{
			colors[i] = waterGradient.Evaluate((float)i/count);
		}

		RampTexture.SetPixels(colors);
		RampTexture.Apply();
		
		// 全局赋值，将 RampTexture 纹理赋值到 _RampTexture 上 
		Shader.SetGlobalTexture("_RampTexture", RampTexture);
	}
}
