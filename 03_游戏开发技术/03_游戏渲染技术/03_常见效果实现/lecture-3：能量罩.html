能量罩的实现组成

概念：能量罩的实现组成

  • 概念：
    • 一个逼真的能量罩效果通常由三个基本视觉元素构成，它们共同作用，模拟出一种半透明、发光且能与环境交互的力场；
  • 概念：​边缘光（菲涅尔效应）​
    • 其原理是，当视线方向与物体表面法线接近垂直（即看向物体的边缘）时，反射会增强；
    • 在 Shader 中，我们用 1 - saturate(dot(normal, viewDir)) 来计算，并对结果进行幂运算（pow）来控制边缘光的宽度和衰减；
  • 概念：交界处高亮
    • 这是让能量罩与场景产生交互的关键，它使能量罩在与其它物体接触时产生明亮的辉光，暗示其具有实体并正在发生碰撞或遮挡；
    • 交界处高亮的本质是对比当前能量罩像素的深度与场景中实际存在物体的深度；
  • 概念：动态表面纹理
    • 为了表现能量罩表面的能量流动，通常会采样一张蜂窝状或网格状的纹理，并让其随时间偏移；
    • 例如：i.uv + float2(0, _Time.y * 0.3)，从而营造出能量流动的效果；

交界处高亮实现原理

概念：交界处高亮实现原理

  • 概念：
    • 交界处高亮的本质是对比当前能量罩像素的深度与场景中实际存在物体的深度；
    • 深度图是一张特殊纹理，它存储了从相机视角出发，场景中每个像素位置所见到的最浅（对于不透明物体）物体的深度值（Z 值），在 URP 中，你需要确保在 URP Asset 中启用了深度纹理（Depth Texture），才可以在 Shader 中通过 _CameraDepthTexture 来采样获取；
  • 实现：
    • （1）将能量罩设置为半透明：
      • 能量罩 Shader 的渲染队列（Queue）设置为 "Transparent"。这样做有两个重要原因：
        1. ​防止深度写入​：半透明物体通常关闭深度写入（ZWrite Off），这样它就不会覆盖深度缓冲区中原本物体的深度信息，保证了 _CameraDepthTexture 中记录的是场景中不透明物体的深度，而非能量罩本身的深度；
        1. 正确的渲染顺序​：确保能量罩在所有不透明物体渲染完毕后再进行渲染，这样才能正确采样到包含场景完整信息的深度图；
    • （2）获取能量罩表面的深度信息：​
      • 在能量罩的片元着色器（Fragment Shader）中，你需要计算当前像素在观察空间（View Space）​​ 下的深度值 Z_shield；
      • 注意：这里的 o.positionVS = TransformWorldToView(positionWS); 计算得到的观察空间下的 Z 值是一个负值，因为 Unity 当中朝向 Z 轴方向是负方向；
    • （3）采样场景深度图​：
      • 使用当前像素在屏幕空间中的坐标来采样 _CameraDepthTexture，得到的是非线性深度值，需要转换到线性深度 Z_scene 才便于计算。比如下述代码：
    • （4）深度比较与高亮生成​：
      • 这是最核心的一步，通过比较 Z_scene 和 （因为 为负，比较时通常取绝对值或处理为正数），在能量罩单独存在的部分， 等于远处背景的深度，远大于能量罩自身的 |Zshield|。而当能量罩与其他物体（如 Cube）相交时，在相交区域，$Z{scene}$ 记录的是 Cube 的深度，这个值会非常接近甚至小于 |Z_shield|（因为 Cube 在能量罩前面）
      • 当结果远大于 0 时，说明场景物体在能量罩后方，不是交界处；当结果接近 0 时，就说明能量罩表面与场景中的物体表面非常接近，此处就是我们要高亮的交界处；
      • 生成高亮​：利用 saturate(1.0 - depthDifference) 或类似的平滑函数，将接近 0 的深度差映射为高亮强度（接近 1），而差值大的区域强度为 0
    • （5）高亮过渡与高亮颜色：

      • 将高亮乘以高亮衰减系数，然后叠加上高亮区域的颜色，得到最终效果；

Shader "Unlit/EnergyShield"
{
    Properties
    {
        _BaseColor("BaseColor",Color) = (1,1,1,1)
        _BaseMap("BaseMap",2D) = "white"{}
        _HighLightFade("HighLightFade",Float) = 3
        _HighLightColor("HighLightColor",Color) = (1,1,1,1)
    }
    
    SubShader
    {
        Tags
        {
            "RenderType"="Transparent"
            "Queue" = "Transparent"
            "RenderPipeline" = "UniversalPipeline"
            "IgnoreProjector" = "True"
        }
        LOD 100

        Pass
        { 
            Name "SimplesUnlit"
            HLSLPROGRAM
            #pragma vertex vert
            #pragma fragment frag
            #pragma multi_compile_fog
            #include "Packages/com.unity.render-pipelines.core/ShaderLibrary/Color.hlsl"
            #include "Packages/com.unity.render-pipelines.universal/ShaderLibrary/Core.hlsl"
            #include "Packages/com.unity.render-pipelines.universal/ShaderLibrary/Lighting.hlsl"
            #include "Packages/com.unity.render-pipelines.universal/ShaderLibrary/ShaderGraphFunctions.hlsl"

            struct Attributes
            {
                float4 positionOS : POSITION;
                float2 uv : TEXCOORD0;
            };

            struct Varyings
            {
                float4 positionCS : SV_POSITION;
                float2 uv : TEXCOORD0;
                float fogCoord : TEXCOORD1;
                float3 positionVS : TEXCOORD2;
            };

			// 变量定义在 CBuffer 当中
            CBUFFER_START(UnityPerMaterial)
	            half4 _BaseColor;
	            half _HighLightFade;
	            half4 _HighLightColor;
            CBUFFER_END
            
            TEXTURE2D(_BaseMap);SAMPLER(sampler_BaseMap);float4 _BaseMap_ST;
            TEXTURE2D(_CameraDepthTexture);SAMPLER(sampler_CameraDepthTexture);

            Varyings vert (Attributes v)
            {
                Varyings o;
                
                // 为了得到观察空间坐标，先将其转化为世界空间
                float3 positionWS = TransformObjectToWorld(v.positionOS.xyz);
                
                // 将顶点从世界空间转化为视图空间
                o.positionVS = TransformWorldToView(positionWS);
                
                // 将顶点从视图空间转化为齐次裁剪空间
                o.positionCS = TransformWViewToHClip(o.positionVS);
                
                o.uv = TRANSFORM_TEX(v.uv,_BaseMap);
                o.fogCoord = ComputeFogFactor(o.positionCS.z);

                return o;
            }

            half4 frag (Varyings i) : SV_Target
            {
                // 屏幕空间 UV
                float2 screenUV = i.positionCS.xy / _ScreenParams.xy;
                // 采样深度图
                half4 depthMap = SAMPLE_TEXTURE2D(_CameraDepthTexture,sampler_CameraDepthTexture,screenUV);
                
                // 获取当前片段对应深度图中像素在观察空间下的Z值
                half depth = LinearEyeDepth(depthMap,_ZBufferParams);
                
                // 假设当前能量罩和其他物体相交，则当前能量罩上被相交 的部分的表面片元对应的深度值应该是更大的
                half4 hightLight = depth + i.positionVS.z;
                
                hightLight *= _HighLightFade;
                half4 hightLightInvers = 1 - hightLight;
                hightLightInvers *= _HighLightColor;
                hightLightInvers += hightLightInvers;
                
                return hightLightInvers;
            }
            
            ENDHLSL
        }
    }
}

{

    • 其中 _CameraOpaqueTexture 是当前抓取的屏幕截图，如下所示：

Shader "Unlit/EnergyShield"
{
    Properties
    {
        _BaseColor("BaseColor",Color) = (1,1,1,1)
        _BaseMap("BaseMap",2D) = "white"{}
        _BaseMapInstensity("BaseMapInstensity",Float) = 0.1
        _HighLightFade("HighLightFade",Float) = 3
        _HighLightColor("HighLightColor",Color) = (1,1,1,1)
        _FresnelInstensity("FresnelInstensity",Range(1,15)) = 8
        _FresnelColor("FresnelColor",Color) = (1,1,1,1)
        
        _Tiling("Tiling",Float) = 4
        _Distort("Distort",Range(0,1)) = 0.4
    }
    
    SubShader
    {
        Tags
        {
            "RenderType"="Transparent"
            "Queue" = "Transparent"
            "RenderPipeline" = "UniversalPipeline"
            "IgnoreProjector" = "True"
        }
        LOD 100
        Blend SrcColor OneMinusSrcColor

        Pass
        { 
            Name "SimplesUnlit"
            HLSLPROGRAM
            #pragma vertex vert
            #pragma fragment frag
            #pragma multi_compile_fog
            #include "Packages/com.unity.render-pipelines.core/ShaderLibrary/Color.hlsl"
            #include "Packages/com.unity.render-pipelines.universal/ShaderLibrary/Core.hlsl"
            #include "Packages/com.unity.render-pipelines.universal/ShaderLibrary/Lighting.hlsl"
            #include "Packages/com.unity.render-pipelines.universal/ShaderLibrary/ShaderGraphFunctions.hlsl"

            struct Attributes
            {
                float4 positionOS : POSITION;
                float2 uv : TEXCOORD0;
                half3 normal : NORMAL;
            };

            struct Varyings
            {
                float4 positionCS : SV_POSITION;
                float4 uv : TEXCOORD0;
                float fogCoord : TEXCOORD1;
                float3 positionVS : TEXCOORD2;
                half3 normalWS : TEXCOORD3;
                half3 viewWS : TEXCOORD4; // 世界坐标下，相机到物体表面的坐标
            };

			// 变量定义在 CBuffer 当中
            CBUFFER_START(UnityPerMaterial)
	            half4 _BaseColor;
	            half _BaseMapInstensity;
	            half _HighLightFade;
	            half4 _HighLightColor;
	            half _FresnelInstensity;
	            half4 _FresnelColor;
	            half _Tiling;
	            half _Distort;
            CBUFFER_END
            
            TEXTURE2D(_BaseMap);SAMPLER(sampler_BaseMap);float4 _BaseMap_ST;
            TEXTURE2D(_CameraDepthTexture);SAMPLER(sampler_CameraDepthTexture);
            TEXTURE2D(_CameraOpaqueTexture);SAMPLER(sampler_CameraOpaqueTexture);

            Varyings vert (Attributes v)
            {
                Varyings o;
                
                // 为了得到观察空间坐标，先将其转化为世界空间
                float3 positionWS = TransformObjectToWorld(v.positionOS.xyz);
                
                // 将顶点从世界空间转化为视图空间
                o.positionVS = TransformWorldToView(positionWS);
                
                // 将顶点从视图空间转化为齐次裁剪空间
                o.positionCS = TransformWViewToHClip(o.positionVS);
                
                // uv 的 zw 用来存储 baseMap 的 UV
                o.uv.zw = TRANSFORM_TEX(v.uv,_BaseMap);
                o.uv.xy = v.uv;
                
                o.fogCoord = ComputeFogFactor(o.positionCS.z);
                
                // 世界空间下的法线
                o.normalWS = TransformObjectToWorldNormal(v.normal);
                
                // 世界坐标下，相机到物体表面的坐标
                o.viewWS = _WorldSpaceCameraPos - positionWS;

                return o;
            }

            half4 frag (Varyings i) : SV_Target
            {
                // 完整颜色
                half4 color;

                // --------------- 蜂窝纹理 ----------------
                half4 baseMap = SAMPLE_TEXTURE2D(_BaseMap,sampler_BaseMap,i.uv.zw);
                color = (baseMap * float(_BaseMapInstensity * abs(sin(_Time.y)))) * _BaseColor;
                color.rgb = MixFog(color.rgb,i.fogCoord);

                // --------------- 交界处高亮 ------------------
                // 屏幕空间 UV
                float2 screenUV = i.positionCS.xy / _ScreenParams.xy;
                // 采样深度图
                half4 depthMap = SAMPLE_TEXTURE2D(_CameraDepthTexture,sampler_CameraDepthTexture,screenUV);
                // 获取当前片段对应深度图中像素在观察空间下的Z值
                half depth = LinearEyeDepth(depthMap,_ZBufferParams);
                // 假设当前能量罩和其他物体相交，则当前能量罩上被相交 的部分的表面片元对应的深度值应该是更大的
                half4 hightLight = depth + i.positionVS.z;
                // 叠加高亮颜色和计算衰减
                hightLight *= _HighLightFade;
                half4 hightLightInvers = 1 - hightLight;
                hightLightInvers *= _HighLightColor;
                hightLightInvers = saturate(hightLightInvers);
                color += hightLightInvers;

                // --------------- 外发光 --------------------
                half3 N = normalize(i.normalWS);
                half3 V = normalize(i.viewWS); 
                half fresnel = pow(max(0,1-dot(N,V)),_FresnelInstensity);
                half4 fresnelColor = _FresnelColor * fresnel;
                color += fresnelColor;

                // --------------- 蜂窝纹理扭曲 ----------------
                // 当前帧的抓屏
                // 扭曲的UV，在屏幕空间的UV喝baseMap作为扭曲UV之间插值
                half4 baseMap01 = SAMPLE_TEXTURE2D(_BaseMap,sampler_BaseMap,i.uv.zw + float2(0,_Time.y));
                float2 distortUV = lerp(screenUV,baseMap01.rr,_Distort);
                half4 opaqueTex = SAMPLE_TEXTURE2D(_CameraOpaqueTexture,sampler_CameraOpaqueTexture,distortUV);
                half4 distort = half4(opaqueTex.xyz,1);
                half flowMask = frac(i.uv.y * _Tiling + _Time.y);
                distort *= flowMask;
                color += distort;
                
                return color; 
            }
            
            ENDHLSL
        }
    }
}
