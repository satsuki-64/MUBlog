ToLua 调用 C# 枚举

概念：ToLua 调用 C# 枚举

  • 代码：
public enum E_MyEnum
{
	Idle,
	Move,
	Atk
}
  • 代码：
-- 调用Unity自带的枚举
-- 枚举调用规则和类的调用规则是一样的
-- 命名空间.枚举名.枚举成员
-- 也支持取别名
PrimitiveType = UnityEngine.PrimitiveType
GameObject = UnityEngine.GameObject
-- 使用GameObject中的静态方法创建几何体
local obj = GameObject.CreatePrimitive()

-- 调用自定义的枚举
-- 注意也需要加入类型名
local c = E_MyEnum.Idle
local d = E_MyEnum.Idle

-- userdata类型，保留了语言中的数据类型
if c==d then 
	print(111) -- 这一段代码不会打印，因为枚举不相等
else
	print(222) -- 输出出来，因为枚举不相等
end 

-- 枚举转各种类型
print(tostring(c))
print(c:ToInt())
print("数字转枚举："..E_MyEnum.IntToEnum(0))  -- 将数字0转为枚举Idle
注意：使用枚举
  • 在 ToLua 中使用 C# 的枚举，也需要在 CustomSetting 的 customTypeList 中加入类型名。然后添加之后，也需要点击生成 Lua 代码；
  • 和 XLua 的区别，就是没有提供字符串转枚举的功能；
  • 注意即使是同一个枚举，它们互相之间比较也是不相等的；

ToLua 调用 ref 和 out 方法

概念：ToLua 调用 ref 和 out 方法

  • 代码：
public class Test
{
	public int RefFun(int a,ref int b,ref int c,int d)
	{
		// ...
	}
	
	public int OutFun(int a,out int b,out int c,int d)
	{
		// ...
	}
	
	public int RefOutFun(int a,out int b,ref int c)
	{
		// ...
	}
}
  • 代码：
-- 如果是out和ref函数，那么第一个返回值是函数的默认返回值（如果没有则不用管）。之后的返回值，就是ref和out从左到右对应的结果
local obj = Test()
print(obj:RefFun(10,0,0,1))
local a,b,c = obj:RefFun(100,1,1,0) -- a是返回值，b和c是两个ref参数b和c

-- out
print(obj:OutFun(20,39)) -- 会发生报错
print(obj:OutFun(20,nil,nil,39)) -- 不会发生报错

-- ref、out一起
print(obl:RefOutFun(20,nil,1))
总结：使用 ref 和 out 方法
    1. toLua 和 xLua 中对 ref 和 out 函数使用基本类似，通过多返回值的形式来处理的；
    • 如果是 out 和 ref 函数，那么第一个返回值是函数的默认返回值（如果没有则不用管）；
    • 之后的返回值，就是 ref 和 out 参数的从左到右对应的结果；
    1. toLua 中的 out 函数调用和 xLua 中有区别，xLua 中可以省略传入 out 参数，toLua 中不能省略；

ToLua 调用重载函数

概念：ToLua 中调用重载函数

  • 代码：
public class Test
{
	public int Calc(int a)
	{
		Debug.Log("Int");
		return a;
	}
	
	public float Calc(float a)
	{
		Debug.Log("Float");
		return a;
	}
	
	public string Calc(string a)
	{
		Debug.Log("String");
		return a;
	}
	
	public int Calc(int a,int b)
	{
		Debug.Log("Int a+b");
		return a+b;
	}
	
	public int Calc(int a,out int b)
	{
		b = 10;
		return a+b;
	}
}
  • 代码：
local obj = Test()
print(obj:Calc())
print(obj:Calc(1)) -- 这里调用了Float的重载
print(obj:Calc(1.2)) -- 调用的也是Float的重载
print(obj:Calc("123")) -- 输出的结果正常

-- 这里调用的是非out参数的重载
print(obj:Calc(10,2))

-- 在ToLua中调用Out重载有一个固定的套路，直接传空
-- 这意味着在使用Unity中的一些api有out时，可以直接使用nil占位
print(obj:Calc(10,nil))
总结：ToLua 与重载函数
    1. ToLua 和 XLua 一样，对重载函数的精度支持不好，因为 lua 中只有 number 一种类型，lua 会分不清楚、全部使用精度高的类型，比如 Int 数值进入 Lua 后变成 Float；
    1. ToLua 中用 Unity 中的一些 api 有 out 时，可以直接使用 nil 占位；
    1. ToLua 对于 ref 的支持不好，建议在 ToLua 中只使用 out、不使用 ref，特别是在重载时；

ToLua 调用委托和事件

概念：ToLua 调用委托和事件

  • 概念：
public class Test
{
	public UnityAction del;
	public event UnityAction eventAction;
	
	public void DoDel()
	{
		if(del is not null)
		{
			del();
		}
	}
	
	public void DoEvent()
	{
		if(eventAction is not null)
		{
			eventAction();
		}
	}
	
	public void ClearEvent()
	{
		eventAction = null;
	}
}
  • 代码：
local obj = Test()
-- 使用C#中的委托，就是用来装lua函数
local fun = function()
	print("Lua函数Fun")
end

obj.del = fun
obj.del = obj.del + fun
obj.del = obj.del + function()
	print("临时声明的函数")
end

-- 报错：obj.del()
-- 这种执行的方法才是正确的
obj:DoDel()

-- 委托中减去函数
obj.del = obj.del - fun
obj.del = obj.del - fun
obj:DoDel()

-- 委托中清空函数
obj.del = nil
obj:DoDel() -- 无反应

-- 事件
local fun2 = function
	print("事件加的函数")
end

-- 加减函数
-- bj.eventAction = fun2：这个代码如果直接执行会报错，因为按照C#中的规则，事件直接“=”会发生报错，只能"+="或者"-="
obj.eventAction = obj.eventAction + fun2
obj.eventAction = obj.eventAction + function()
	print("你好")
end

-- 执行事件
obj:DoEvent()

-- 清空事件
obj:ClearEvent()
obj:Event() -- 无反应
总结：关于 ToLua 调用委托和事件
    1. Lua 中没有复合运算符，不能使用 +=。并且如果是第一次往委托里面加函数，因为是 nil 不能直接 +，因此第一次需要使用等于号 =；
    1. xLua 中调用委托可以直接执行 obj.del()，但是在 toLua 中不行，其必须得调用 C# 中执行委托的函数；
    1. xLua 中事件加函数使用：对象:事件名("+/-",函数)，但是 toLua 中事件加减函数的方式和委托方式一样；

ToLua 调用协程

概念：ToLua 调用协程

  • 代码：
local coDelay = nil

StartDelay = function()
	-- toLua提供的开启协程的方法
	-- Unity当中的开启协程的方法
	coDelay = StartCoroutline()
end

Delay = function()
	local c = 1
	while true do
		-- 在ToLua中可以直接使用Unity中的协程相关的方法
		WaitForSeconds(1)
		
		-- 比如：Yield(0)、WaitForFixedUpdate()、WaitForEndOfFrame()、Yield(异步加载返回值)
		print("Count"..c)
		c = c + 1
		
		if c > 5 then
			StopDelay()
			break
		end
	end
end

-- 释放协程
StopDelay = function()
	StopCoroutline(coDelay)
	coDelay = nil
end

-- 使用协程之前，需要在ToLua中进行登记
StartDelay()
