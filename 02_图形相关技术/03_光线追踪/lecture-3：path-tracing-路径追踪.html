间接光照渲染计算

概念：间接光照渲染的蒙特卡洛积分形式

  • 分析：
    • 光线在 P 点接收直接光照后，此时光线可以反射到 Q 点；
    • 但如果 Q 点是 Diffuse 时，此时光线还可以再从 Q 点反射到 P 点；
  • 图示：
  • 代码：支持全局光照的算法
  • 代码：详细版
# 定义着色函数：计算点 p 在出射方向 wo 上的辐射亮度
def shade(p, wo):
    # 初始化输出辐射亮度（Lo = L_out）
    Lo = 0.0
    
    # 随机采样 N 个入射方向 wi（重要性采样，概率密度函数为 pdf）
    N = 1024  # 采样数示例
    for i in range(N):
        wi = sample_direction(p, pdf)  # wi ~ pdf(wi) 按概率密度采样方向
        
        # 追踪从 p 点出发的 wi 方向光线
        hit_object, r = trace_ray(p, wi)
        
        if hit_object == "LIGHT":
            # 情况1：光线直接击中光源
            L_i = get_light_radiance(r.hit_point)  # 获取光源亮度
            fr = BRDF(p, wi, wo)                  # 双向反射分布函数
            cosine = dot(normal_at(p), wi)         # 法向量与入射角余弦
            
            # 蒙特卡洛积分累加（直接光照贡献）
            Lo += (1/N) * L_i * fr * cosine / pdf(wi)  # 图中蓝色公式部分
            
        elif hit_object == "OBJECT":
            # 情况2：光线击中物体（间接光照）
            q = r.hit_point                        # 击中点坐标
            fr = BRDF(p, wi, wo)                   # BRDF 值
            cosine = dot(normal_at(p), wi)         # 余弦项
            
            # 递归计算间接光照（shade(q, -wi) 为图中红色突出部分）
            indirect_light = shade(q, -wi)          # 递归调用
            
            # 蒙特卡洛积分累加（间接光照贡献）
            Lo += (1/N) * indirect_light * fr * cosine / pdf(wi)
    
    return Lo  # 返回最终计算的辐射亮度
  • 分析：
    • 其中分为两种情况，如果是打到光源，就直接使用 L_i 进行计算，如果打到的是其他物体，那么就递归的进行计算 shade(q,-wi)；
    • 其中 shade(q,-wi) 的 -wi 方式当前光线的反方向；

路径追踪的问题

问题一：光线爆炸

概念：光线数量爆炸

  • 概念：
    • 光线会爆炸，光线的数量太多；
    • 当 N 等于多少的时候，这个指数不会爆炸？
    • 只有当 N=1 的时候，这个指数不会爆炸。因为每次只反射出去一根光线；
    • 而当 N=1 时，这种光线追踪的技术就被称之为路径追踪；
  • 图示：
  • 代码：
# 光线追踪着色函数
def shade(p, wo):
    # 随机选择 ONE 个方向 wi（根据概率密度函数 pdf(w)）
    wi = random_choose_direction(pdf)  # 突出"ONE"表示单次采样
    
    # 从点 p 向方向 wi 发射光线
    ray = trace_ray(p, wi)
    
    if ray.hit_type == "LIGHT":
        # 情况1：光线命中光源
        L_i = ray.get_light_radiance()      # 获取光源亮度
        fr = BRDF(p, wi, wo)                # 双向反射分布函数
        cosine = dot(normal(p), wi)         # 法向量与入射方向点积
        
        # 返回直接光照贡献（严格对应图中公式）
        # 这当中没有乘以 1/N 的部分了
        return L_i * fr * cosine / pdf(wi)  # 黑色文字部分
        
    elif ray.hit_type == "OBJECT":
        # 情况2：光线命中物体（q为命中点）
        q = ray.hit_point
        fr = BRDF(p, wi, wo)                # BRDF项
        cosine = dot(normal(p), wi)         # 余弦项
        
        # 这当中没有乘以 1/N 的部分了
        # 递归计算间接光照（对应图中红色逻辑）
        return shade(q, -wi) * fr * cosine / pdf(wi)
  • 图示：
    • 这些所有的路径都会穿过这些像素，然后反射时只会随机选择一个方向进行反射；
    • 每个像素发射一条光线；

代码：光线生成

# 光线生成函数
def ray_generation(camPos, pixel):
    # 在像素内均匀选取N个采样位置（图中第一行说明）
    N = 16  # 示例采样数
    samples = uniformly_sample_pixel(pixel, N)
    
    # 初始化像素辐射度为0（图中第二行）
    pixel_radiance = 0.0
    
    # 对每个采样点进行处理（图中第三行）
    for sample in samples:
        # 发射从相机到采样点的光线（图中第四行）
        ray_direction = normalize(sample - camPos)
        r = Ray(camPos, ray_direction)
        
        # 检测光线是否命中场景（图中第五行）
        hit_result = scene.intersect(r)
        if hit_result.hit:
            p = hit_result.point  # 命中点坐标
            
            # 计算采样点到相机的方向（图中第六行隐含逻辑）
            sample_to_cam = normalize(camPos - p)
            
            # 累加着色结果（图中第六行公式）
            # 注意：图片中疑似有拼写错误（heralv/se1→应为radiance/1）
            pixel_radiance += (1/N) * shade(p, sample_to_cam)
    
    # 返回最终辐射度（图中最后一行）
    # 注意：图片中"pixel_length_length"应为"pixel_radiance"
    return pixel_radiance

问题二：递归问题

概念：递归方程的递归条件

  • 概念：
    • 递归方程必须要有终止的条件；
    • 但是真实世界当中的光线是没有停止弹射的，而进行光线传播计算时不能模拟弹射无数次；
    • 因此引入了一种方法：俄罗斯轮盘赌；
    • 用这个 俄罗斯轮盘赌 的方法，以一定的概率、在一定的条件下停止继续弹射；
  • 概念：俄罗斯轮盘赌 的方法
    • 以一定的概率 P，朝某个方向打出一条光线，而以 1-P 的概率不打出方向；
    • 在这种方式下，期望的结果为 E=P*(Lo/P)+(1-P)*O=Lo，最后的结果就是 L0，这样的结果就是正确的结果；
  • 代码：
# 着色函数（严格还原图片内容）
def shade(p, wo):
    # 手动指定俄罗斯轮盘赌概率（图中第二行）
    P_RR = 0.8  # 示例值，图中未指定具体数值
    
    # 从[0,1]均匀分布随机选取ksi（图中第三行）
    ksi = random.uniform(0, 1)
    
    # 俄罗斯轮盘赌终止条件（图中第四行）
    if ksi > P_RR:
        return 0.0  # 终止光线追踪
    
    # 随机选择单个方向wi（根据pdf(w)分布）（图中第五行）
    wi = sample_direction(pdf)
    
    # 追踪光线（图中第六行）
    ray = trace_ray(p, wi)
    
    if ray.hit_type == "LIGHT":
        # 命中光源的情况（图中第七行）
        L_i = ray.get_light_radiance()
        fr = BRDF(p, wi, wo)
        cosine = dot(normal(p), wi)
        
        # 严格对应图中公式（注意红色P_RR）
        return L_i * fr * cosine / pdf(wi) / P_RR
        
    elif ray.hit_type == "OBJECT":
        # 命中物体的情况（图中第八行）
        q = ray.hit_point
        fr = BRDF(p, wi, wo)
        cosine = dot(normal(p), wi)
        
        # 递归调用（注意红色-wi和P_RR）
        return shade(q, -wi) * fr * cosine / pdf(wi) / P_RR

def shade(p, wo):
    # ======================
    # 直接光照贡献（光源采样）
    # ======================
    # 在光源表面均匀采样点x'（pdf_light = 1/A，A为光源面积）
    x_prime, pdf_light = uniform_sample_light()
    
    # 计算直接光照贡献（严格对应图中公式）
    L_dir = (L_i * f_r * cosθ * visibility) / (distance(x_prime,p)^2 * pdf_light)
    # 其中：
    # L_i: 光源亮度
    # f_r: BRDF值
    # cosθ: 着色点法向量与光照方向的夹角余弦
    # visibility: 可见性测试（图中方块符号表示）
    
    # ======================
    # 间接光照贡献（半球采样）
    # ======================
    L_indir = 0.0
    
    # 俄罗斯轮盘赌测试（概率P_RR）
    if random() < P_RR:  # 图中未指定具体P_RR值
        # 在半球上均匀采样方向wi（pdf_hemi = 1/2π）
        wi, pdf_hemi = uniform_sample_hemisphere()
        
        # 发射探测光线
        ray = trace_ray(p, wi)
        if ray.hits_non_emitting_object_at(q):
            # 递归计算间接光照（严格对应图中公式）
            L_indir = shade(q, -wi) * f_r * cosθ / pdf_hemi / P_RR
    
    # 返回总光照贡献
    return L_dir + L_indir
