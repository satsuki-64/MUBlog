概念：质点弹簧系统 - Mass Spring System

  • 概念：
    • 质点弹簧系统是一系列相互连接的质点和弹簧；
    • 质点弹簧系统是计算机图形学、物理仿真和工程力学中常用的简化模型，用于模拟弹性体​（如布料、绳索、软体组织）的动力学行为；
    • 其核心思想是将物体离散化为质点（质量点）​和弹簧（弹性连接）​的组合，通过力学方程驱动运动；
  • 概念：常见连接结构

有限元系统

基础概念

概念：有限元系统

  • 概念：
    • 将物体离散化为有限个单元​（如四面体、六面体），通过求解连续介质的力学方程（如线性/非线性弹性方程）模拟变形；
    • 其基于连续介质力学​（应力-应变关系），使用形函数（Shape Functions）​插值单元内部的位移场；
  • 特点：
    • 高精度：严格遵循物理定律，能模拟复杂材料（各向异性、塑性变形）；
    • 计算复杂：需组装和求解大型稀疏矩阵，计算成本高；

粒子系统基础概念

概念：粒子系统

  • 概念：
    • 通过简单规则驱动大量粒子，通过集体行为呈现复杂视觉效果；
  • 组成：
  • 实现：
      1. 初始化粒子
      • 内存预分配：创建固定容量的粒子对象池，避免运行时频繁内存分配；
      • 发射器设置：定义粒子的生成位置（如点、区域或模型表面）、发射速率（每秒生成粒子数）、初始属性范围（速度、大小、颜色的随机区间）；
      • 力场初始化：加载影响粒子运动的外部力（如重力、风力、涡流场）；
      1. 粒子生成（发射）
      • 速率控制：根据帧时间（delta_time）和发射速率计算当前帧应生成的粒子数量；
        • 例如，若发射速率为 100个/秒，上一帧耗时 0.02 秒，则生成 2 个新粒子；
      • 属性随机化：为每个新粒子赋予初始属性；
      1. 粒子更新
      • 每帧遍历所有存活粒子，更新其状态；
      • 位置更新：根据当前速度与时间步长，采用欧拉积分计算新位置；
      • 年龄累加：粒子年龄（age）随时间增长，当超过预设生命周期时标记为待销毁；
      1. 粒子消亡
      • 移除无效粒子：将已超过生命周期或碰撞后应消失的粒子移出活跃列表；
      • 对象池回收：将消亡粒子的内存重新放入池中，供后续新粒子复用，避免内存碎片
      1. 渲染

// 粒子
struct Particle {
    glm::vec3 position;  // 位置
    glm::vec3 velocity;   // 速度
    glm::vec4 color;      // 颜色 (RGBA)
    float size;           // 大小
    float life;           // 剩余生命周期
    float maxLife;        // 总生命周期
};

// 粒子发射器
class ParticleEmitter {
public:
    glm::vec3 position;           // 发射位置
    float rate;                   // 每秒发射粒子数
    glm::vec3 velocityMin;        // 最小初始速度
    glm::vec3 velocityMax;        // 最大初始速度
    glm::vec4 colorStart;         // 初始颜色
    glm::vec4 colorEnd;           // 结束颜色
    float sizeStart, sizeEnd;     // 大小变化范围
    float lifeMin, lifeMax;       // 生命周期范围

    // 生成新粒子
    Particle spawnParticle() {
        Particle p;
        p.position = position;
        p.velocity = glm::linearRand(velocityMin, velocityMax);
        p.color = colorStart;
        p.size = sizeStart;
        p.life = glm::linearRand(lifeMin, lifeMax);
        p.maxLife = p.life;
        return p;
    }
};

// 粒子系统
class ParticleSystem {
private:
    std::vector<Particle> particles;  // 粒子池
    ParticleEmitter emitter;          // 发射器
    GLuint VAO, VBO;                  // OpenGL缓冲区

public:
    ParticleSystem() {
        initRenderer();
        setupEmitter();
    }

    // 初始化OpenGL渲染数据
    void initRenderer() {
        glGenVertexArrays(1, &VAO);
        glGenBuffers(1, &VBO);
        glBindVertexArray(VAO);
        glBindBuffer(GL_ARRAY_BUFFER, VBO);
        glBufferData(GL_ARRAY_BUFFER, sizeof(Particle) * 1000, nullptr, GL_DYNAMIC_DRAW);

        // 位置属性
        glEnableVertexAttribArray(0);
        glVertexAttribPointer(0, 3, GL_FLOAT, GL_FALSE, sizeof(Particle), (void*)offsetof(Particle, position));
        
        // 颜色属性
        glEnableVertexAttribArray(1);
        glVertexAttribPointer(1, 4, GL_FLOAT, GL_FALSE, sizeof(Particle), (void*)offsetof(Particle, color));
        
        // 大小属性
        glEnableVertexAttribArray(2);
        glVertexAttribPointer(2, 1, GL_FLOAT, GL_FALSE, sizeof(Particle), (void*)offsetof(Particle, size));
        
        glBindVertexArray(0);
    }

    // 配置发射器参数
    void setupEmitter() {
        emitter.position = glm::vec3(0.0f);
        emitter.rate = 50.0f;
        emitter.velocityMin = glm::vec3(-1.0f, 2.0f, -1.0f);
        emitter.velocityMax = glm::vec3(1.0f, 5.0f, 1.0f);
        emitter.colorStart = glm::vec4(1.0f, 0.5f, 0.0f, 1.0f); // 橙色
        emitter.colorEnd = glm::vec4(1.0f, 0.0f, 0.0f, 0.0f);   // 淡出到透明红
        emitter.sizeStart = 0.2f;
        emitter.sizeEnd = 0.0f;
        emitter.lifeMin = 1.0f;
        emitter.lifeMax = 3.0f;
    }

    // 更新所有粒子状态
    void update(float deltaTime) {
        // 生成新粒子
        int newParticles = static_cast<int>(emitter.rate * deltaTime);
        for (int i = 0; i < newParticles; ++i) {
            if (particles.size() < 1000) { // 限制最大粒子数
                particles.push_back(emitter.spawnParticle());
            }
        }

        // 更新现有粒子
        for (auto& p : particles) {
            p.life -= deltaTime;
            if (p.life > 0.0f) {
                p.velocity += glm::vec3(0.0f, -9.8f, 0.0f) * deltaTime; // 重力
                p.position += p.velocity * deltaTime;
                p.color = glm::mix(emitter.colorEnd, emitter.colorStart, p.life / p.maxLife); // 颜色插值
                p.size = glm::mix(emitter.sizeEnd, emitter.sizeStart, p.life / p.maxLife);    // 大小插值
            }
        }

        // 移除死亡粒子
        particles.erase(
            std::remove_if(particles.begin(), particles.end(),
                [](const Particle& p) { return p.life <= 0.0f; }),
            particles.end()
        );
    }

    // 渲染粒子（使用OpenGL）
    void render() {
        glBindBuffer(GL_ARRAY_BUFFER, VBO);
        glBufferSubData(GL_ARRAY_BUFFER, 0, particles.size() * sizeof(Particle), particles.data());
        glBindVertexArray(VAO);
        glDrawArrays(GL_POINTS, 0, particles.size());
        glBindVertexArray(0);
    }
};

粒子运动

概念：粒子运动 - 欧拉方法

  • 概念：欧拉方法
      1. 在粒子运动的范围中建立速度场
      1. 使用欧拉方法，在每一个时间 t 上计算
      • 下一帧的位置等于上一帧的位置加上时间间隔乘以上一时刻的速度；
      • 即：所有量都是使用上一时刻计算出来；
  • 问题一：不稳定
    • 每一次的时间间隔越大，就越不准确，时间间隔越小，就越精确；
    • 但时间越来越长，其稳定性很差；
  • 问题二：螺旋形速度场
    • 在螺旋形运动场中，会逐渐飞出去；
    • 因此还是不稳定；

概念：粒子运动 - 其他稳定方法

  • 概念：中点法
    • 图示：
    • 公式：
    • 方法：
        1. ​先用欧拉法预估粒子在半个时间步长后的位置和速度​（称为“中点”）；
        1. ​基于中点的状态重新计算受力​（如重力、阻力）；
        1. 用中点的受力更新粒子的最终状态；
      • 欧拉法直接用当前时刻的力更新状态，忽略了运动过程中力的变化；而 Midpoint 方法通过“中点”近似力的变化趋势，减少了误差；
  • 概念：后向欧拉方法
    • 图示：
    • 公式：
    • 概念：
      • 后向欧拉法是一种隐式数值积分方法，用于求解微分方程（如粒子运动、物理仿真）；
      • 与显式的前向欧拉法不同，它通过用未来时刻的状态计算当前更新，具有更强的数值稳定性，但计算复杂度更高；
      • 隐式方法通过“未来状态”反向约束，即使 Δt 较大，解也不会发散；
  • 概念：RK 方法
    • 图示：
      • 下一个时刻的位置会使用上 k1~k4 的值；
    • 概念：
      • Runge-Kutta（龙格-库塔）方法是数值分析中用于求解常微分方程（ODEs）​的一类高精度算法，其中 RK4（四阶龙格-库塔是最常用的版本；
      • 通过多步加权平均来逼近微分方程的解，比欧拉法（单步）更精确：
    • 概念：RK4
      • RK4 通过四个斜率估计值加权平均，在计算量和精度之间达到最优平衡；
      • RK4 通过组合四个不同位置的斜率（起点、两个中点、终点），模拟泰勒展开的四阶精度，显著减少截断误差；

  • 概念：质点法
    • 将物体离散化为一系列质点（Particles）​，每个质点携带质量、位置、速度等属性，物理量（如力、速度）直接在质点上计算，无需依赖固定网格；

  • 概念：网格法
    • 将空间划分为固定网格（Grid）​，物理量（如速度、压力）存储在网格节点或单元中心，通过求解偏微分方程（如 Navier-Stokes）更新物理量；
    • 常见系统就是有限元法；
