反走样的技术

概念：反走样的技术

  • 技术一：提高采样率
    • 即：提高屏幕的像素个数，使得采样率增高；

    • 概念：
      • 模糊就是低通滤波，即将信号的高频的信号去掉，然后再对信号进行采样；
      • 比如：
      • 如果先对原始信号进行模糊，因此将信号的部分砍掉后，此时变成了梯形的形状，信号之间存在间隔；

    • 实现：如何实现滤波操作
      • 用一个一定大小的低通滤波器对图形进行卷积，即可将高频信号去除；
      • 因为卷积就是平均，因此将信号平均后，高频信号被去除；

MSAA

概念：MSAA 技术

  • 引入：
    • 按照上面的先模糊再采样的方法，需要对一个三角形内部的每个像素都做卷积。这样的实现需要将一个三角形内覆盖的区域算出来然后卷积，实现起来比较困难；
    • 因此提出了 MSAA 的技术；
  • 概念：MSAA 技术
    • 假设将一个像素看成很多个小的像素，每一个小的像素有一个小的中心，然后将这些小像素来使用覆盖区域判断，如果小像素覆盖的多，则将其看成相对准确的近似；
    • 即：MSAA 是通过更多的样本，来模拟反走样的第一步，也就是模糊的过程；
  • 图示：MSAA 效果
  • 分析：
    • 通过增加了采样点，并没有得到更多的像素，仅仅只是为了得到更好的像素覆盖；
  • 问题：MSAA 的代价
    • 增大了计算量 -> 2*2：增大了二倍，4*4：增大了四倍；

FXAA 和 TAA

概念：FXAA

  • 先生成有锯齿的图片，生成之后、通过图像匹配，将这个图片有锯齿的边界找到，并且将这些边界换成没有锯齿的边界；

概念：TAA

  • 先找到上一帧的信息，然后用不同帧上、复用上一帧计算的结果，相当于将 MSAA 对于的样本分布在时间上；

概念：可见性

  • 当前场景中有很多的物体，这些物体渲染在二维的图像上时应当具有先后顺序，使得它们能够具有正确的遮挡关系；

Z-Buffer

概念：深度缓冲 - Z-Buffer

  • 概念：
    • 对于每一个物体难以处理其先后关系，但是对于每一个像素，则可以分析出来其先后关系；
    • 每个像素、可以在这个像素内部储存其最近的距离；
    • 因此：在生成每一个图像的同时，还需要生成另一个图像，这个图像只存储深度信息；
    • 即：生成 frame buffer（帧缓冲） 和 depth buffer（深度缓冲）；
  • 图示：
  • 分析：
    • 图示：
    • 过程：
      • 假设一开始所有的像素的深度都是无限大的；
      • 然后可以先把全是 5 的三角形渲染上去；
      • 然后如果有了一个新的三角形，可以将其光栅化时、对每一个覆盖了的像素，判断其 z 的大小，如果更小的、渲染上去；
  • 代码：深度缓冲代码
Initialize depth buffer to ∞
During rasterization:
	for(each triangle T)
		for(each sample(x,y,z) in T)
			if(z<zbuffer[x,y]) // 采样像素的深度比之前的更小，因此更改其深度
			{
				framebuffer[x,y] = z; // 更新颜色
				zbuffer[x,y] = z; // 更新深度
			}
		else ...
  • 概念：Z-Buffer 的复杂度
    • 如果对于 n 个三角形，就有 n 的复杂度；
    • 每次只需要设置最小值，因此不需要排序，因此就是 的复杂度；
    • 并且深度缓冲算法和顺序没有关系。无论先画哪个三角形，最后得到的深度图都是一样的；
